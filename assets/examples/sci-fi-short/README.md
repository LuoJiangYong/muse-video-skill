## 创意 Brief

「最后的记忆贩」是一支 90 秒赛博朋克概念短片，向 BR2049 深度致敬。故事设定在 2089 年尘橙天空下的洛杉矶废墟——一个记忆贩子在回收和贩卖他人记忆的日常中，偶然发现了一段不该存在的记忆片段：关于绿色、树木、未被污染的蓝天。这段记忆打破了他对世界的认知。

**核心视觉锚点**：BR2049 的尘橙天空 + 巨物美学 + 单一光源原则 + 永远有粒子在飘 + 全息投影的「不完美」。

**叙事策略**：全片对白不超过 10 句。用画面、环境、粒子、巨物美学来传递信息。学习 BR2049 的沉默对白、信息释放节奏（每 10 秒一新信息）、和三层反转结构。

## 核心创意决策

1. **8 场叙事结构（三幕）**：世界建立(场1)→角色引入(场2)→任务(场3)→发现异常(场4)→内心冲突(场5)→决断(场6)→后果(场7)→开放结尾(场8)。遵循 BR2049 式信息释放节奏——每 10 秒抛出一个新信息，同时引出新问题。
2. **BR2049 场景色调对照体系**：继承并扩展 BR2049 的 8 色方案——尘橙(室外)/冷蓝灰(室内)/暖金(权力空间)/病态绿(记忆)/褪色金(旧物)/深蓝黑(暗部)/深灰(混凝土)/灰白(结尾)。
3. **角色面部揭示作为叙事手段**：记忆贩 Kael 的面部只在场景 5（第 7 镜头）和场景 7（第 3 镜头）中完全展示——在此之前保持神秘（背影/侧面/暗部）。'看到他的脸' = '看到他的人性觉醒'。
4. **绿色作为视觉稀缺资源**：绿色(#4A9E4A)在全片中只出现不到 5% 的画面——但每次出现都具有极强的叙事冲击力。绿色 = '自然' = '真相' = '希望'（但希望是破碎的）。
5. **4 个结构性静默时刻**：静默不是'没有声音'而是'作为一种声音'——每次静默承担叙事功能，给角色和观众消化信息的空间。
6. **结尾开放性**：记忆贩走向城市边缘的垃圾处理厂（E.V.E. 的方向）——晶片在桌上，故事未完结。学习 BR2049 结尾的 'All the best memories are hers' 的留白美学。

## BR2049 案例引用映射

本案例是 BR2049 的深度实践——以下技法直接继承自 `references/cases/BR2049.md`：

| BR2049 技法 | 本案例应用 |
|-------------|-----------|
| **沉默作为对白**（叙事） | 全片仅 1 句台词（「这不是模拟」），其余通过环境和动作传递信息 |
| **信息释放节奏**（叙事） | 8 场 = 8 个信息单元，每 10 秒新信息+引出新问题（波浪形信息密度） |
| **巨物与微物尺度对比**（镜头） | 场1/4/7 中建筑占画面 70-90%，人在画面中 <5-10% |
| **单一光源原则**（镜头） | 每场戏 1 个主光源——室外=尘橙太阳，室内=冷蓝设备光 |
| **环形构图**（镜头） | 场8 结尾用门框形成矩形框架，记忆贩在框架中远去 |
| **Dolly 克制运动**（镜头） | 全片 60% 镜头为 dolly 极缓运动（速度 ≤ 步行 1/4） |
| **场景色调对照**（色彩） | 8 色方案直接继承 BR2049 6 场景色调对照表 |
| **粒子氛围系统**（特效） | 每个场景至少 1 种粒子——灰烬/尘埃/数据粒子/水雾 |
| **全息投影'不完美'**（特效） | 所有全息元素（广告牌/设备光/E.V.E.投影）均有扫描线+色彩偏移+闪烁 |
| **巨物缓动**（特效） | 飞行车(场1)/机械臂(场4)/垃圾处理厂(场7)均为极缓运动 |
| **工业环境音优先**（声音） | 环境音 60% > 音乐 25% > 对白 10% > 音效 5% |
| **爆发式巨响**（声音） | 场4 低频重击 0.5s → 立刻回静 |
