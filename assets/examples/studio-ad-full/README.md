## 核心创意决策

1. **减法灯光哲学**：学习 BR2049 的单一光源原则——每个场景先确定唯一主光源，其他灯只做补充。广告最大的陷阱是灯光太多，本片反向操作。
2. **材质叙事**：场景 2 的 8 个微距快速剪辑——通过灯光切换展示产品不同材质（拉丝钛合金/蓝宝石玻璃/意大利小牛皮/316L 精钢），每个材质对应一种灯光方案。
3. **沉默为主**：全片仅一句旁白（「光影之间」4 个字），学习 BR2049 的沉默对白技法。画面承载 70% 的信息。
4. **光传递转场**：相邻镜头的光源方向保持一致——光的方向成为视觉连续性线索。
5. **粒子氛围**：金色微尘粒子出现在场景 1 和 5，形成首尾呼应——BR2049 式的「永远有东西在飘」。
