## 叙事技法（→ Writer）

### 动物代言人叙事：猫 = 不需要护照的文化大使

- **表现**：全片 1 分 18 秒，没有出现任何人类面孔（只有戴白手套的手短暂出现）。主角是一只白猫——蓝眼睛，纯白毛色，在卢浮宫的走廊/展厅/楼梯间自由行走。猫不是一个「被带着参观的宠物」——它是「主动的导游」——它自己走、自己看、自己决定去哪。观众不是「跟着一个人参观卢浮宫」——而是「变成一只猫探索卢浮宫」。猫的选择是「非人类中心主义」的——她不在乎《蒙娜丽莎》在哪——她在乎的是地上的光影、墙上的图案纹理、瓷盘的圆、孔雀尾羽的彩虹色。这恰好与展览的主题「图案的奇迹」吻合——展览不是关于「名画」——是关于「图案」——图案是超越人类叙事的存在——猫天生对图案/纹理/光影比人更敏感。
- **手法**：动物代言人的选择逻辑——为什么是猫而不是狗/鸟/人？(a) 猫 = 优雅/独立/安静——匹配卢浮宫和浦东美术馆的「高级艺术」气质；(b) 猫 = 会凝视——猫盯着瓷盘/孔雀/图案时的「凝视」比人类凝视更纯粹（人类凝视 = 「这件值多少钱/谁画的/什么风格」——信息负载；猫凝视 = 「这个图案好看」——纯粹美学）；(c) 猫 = 无国界——猫不需要签证/护照/翻译——她可以从巴黎「跳」到上海只需要一个水面反射的转场——猫的行动在叙事上的自由度远超人类角色；(d) 白猫+蓝眼 = 卢浮宫的建筑色彩（白色石灰岩+蓝色巴黎天空）——猫本身就是卢浮宫的「色彩代言」；橘猫+上海 = 温暖/市井/中国本土——双猫相遇 = 中法文化交流的视觉隐喻。
- **可迁移**：动物代言人选择公式：(a) 动物的「气质」必须与品牌的「气质」一致——奢侈品牌 = 猫/马/豹，科技品牌 = 狗/鸟/鱼；(b) 动物的「身体能力」必须与叙事的「空间自由度」匹配——猫可以跳上窗台/钻进展柜/爬上屋顶（人类不能），鸟可以飞越城市/俯瞰建筑（人类不能），鱼可以在水中「飞翔」；(c) 动物的「无国界性」可以跨越地理/文化/语言的障碍——适合跨境/跨文化的品牌叙事；(d) 动物的「凝视」比人类凝视更纯粹——适合「去知识化/去门槛化」的叙事目标（不需要角色「知道」展览的历史——只需要角色「感受」展览的美）。
- **适用场景**：studio-ad, custom

### 双城蒙太奇：巴黎→上海的「艺术朝圣」反向叙事

- **表现**：全片的叙事结构不是「从上海出发去巴黎看展」——而是「展品从巴黎来到上海」。00:00-00:26 = 巴黎卢浮宫（白猫探索，展品的「原乡」），00:27-00:34 = 打包/盖章/装箱（展品「出发」），00:35-00:42 = 跨洋飞行（插画风格飞机从卢浮宫飞向浦东美术馆），00:43-01:18 = 上海浦东美术馆（白猫抵达，探索新的「家」）。这个方向是反常规的——通常中国观众期待的是「去巴黎看展」的叙事（欧洲中心主义）。但这个宣传片倒转了方向——「卢浮宫来到了上海」——这是对上海观众/中国观众的「文化自信」叙事——「你不用去巴黎——巴黎来到了你身边」。
- **手法**：双城蒙太奇的「方向政治」——(a) 从「文化原乡」（巴黎卢浮宫）→ 到「文化新港」（上海浦东美术馆）——展品的方向 = 文化影响力的方向 = 「从历史到未来、从旧世界到新世界」；(b) 标志性建筑作为「空间坐标」——巴黎的坐标 = 卢浮宫金字塔（当代建筑插入古典宫殿），上海的坐标 = 浦东美术馆玻璃幕墙（当代建筑面对外滩历史建筑群）——两座建筑都是「新+旧」的对话——暗示展览也是「古老图案+当代呈现」的对话；(c) 旅程中段（00:27-00:42）的「打包」段落 = 叙事的「换气」——从巴黎的「艺术沉浸」切换到上海的「艺术抵达」——没有直接硬切——而是用「运输过程」作为叙事缓冲——让观众从巴黎的情绪中「降落」到上海的情绪中。
- **可迁移**：双城蒙太奇的「空间霸权」技法——决定哪个城市是「起点」、哪个是「终点」→ 起点的城市 = 「文化原乡/历史的权威」，终点的城市 = 「文化新港/未来的期待」。起点的画面是「内景」（卢浮宫内部——神圣/封闭/历史），终点的画面是「外景」（浦东美术馆玻璃幕墙——开放/透明/未来）。这个「内→外」的空间运动本身就是叙事——「艺术从封闭的宫殿走向开放的世界」。操作：为双城各选择 1 个「标志性空间」和 1 个「空间性质」（内/外、古/今、封闭/开放）来建立叙事的视觉张力。
- **适用场景**：studio-ad, custom

### 展览即旅程：从「看展品」到「体验旅程」

- **表现**：传统展览宣传片的公式 = 展品特写 + 专家访谈 + 展馆空镜 + 日期/票价。本片的公式 = 一只猫的旅程 — 展品不是「被展示的物品」——而是旅程中的「风景」。展品（瓷盘/壶/碗/织物）在猫的旅途中「自然出现」——猫经过陈列柜时瓷盘在旋转、猫跳上窗台时织物在飘动、猫与孔雀对视时尾羽在绽放。观众不需要「停下来看展品」——展品自己「进入画面」——被整合到猫的「行动流」中。最终效果：观众看完后不会觉得「我看到了一些展品」——而是「我和一只猫一起在卢浮宫和浦东美术馆度过了一段旅程」——这段旅程的情感记忆自动锚定了展览信息。
- **手法**：展览即旅程 = 把「信息」（展品/日期/地点）编码进「体验」（旅程/角色/情感）中。传统模式：信息层（文字/旁白）+ 视觉层（展品/展馆）= 两个分离的层——观众必须「切换注意力」来处理信息。本片模式：信息层被「溶解」进视觉层——展览名称在结尾以「动态插画」的形式出现（不是「字幕」——是「画面的一部分」）——日期/地点被设计成插画中的「视觉元素」而非「阅读材料」。全程观众不需要「切换」到阅读模式——全程在视觉沉浸中。
- **可迁移**：展览/活动宣传的「沉浸替代法」——(a) 确定一个「沉浸载体」——猫/鸟/水滴/光线——任何可以「穿过」展馆空间的主体；(b) 让展品成为「风景」而非「信息」——展品在载体的旅程中自然出现——不是「这是编号 AC-147 的 16 世纪奥斯曼瓷盘」而是「猫从瓷盘前走过——瓷盘的蓝色图案在她白色的毛发上投下影子」；(c) 信息（日期/地点/票价）在结尾以「视觉元素」而非「字幕」的形式出现——观众不需要「读」——只需要「看」。
- **适用场景**：studio-ad

### 跨文化使者：猫作为「无声的翻译」

- **表现**：展览的核心主题是「印度、伊朗、奥斯曼帝国的艺术」——三个文化、三种语言、三种视觉传统——对于中国观众来说可能很陌生。宣传片没有选一个「专家/讲解员」来解释这些文化（那会做成纪录片）。它选了一只猫——猫不需要「理解」印度/伊朗/奥斯曼的区别——猫只需要「感受到」图案的美——而这种感受是人类共通的。猫在卢浮宫伊斯兰展厅中走过——镜头切到孔雀开屏（波斯细密画中常见的视觉母题）、切到瓷盘的蓝色花纹（伊斯兰陶器的标志性特征）、切到织物的金色丝线（奥斯曼纺织品的奢华感）。猫不解释——猫只是「经过」——但她的「经过」本身就是一种「翻译」——「这些来自不同文化的艺术——它们的共同语言是美——而猫是美的纯粹感受者」。
- **手法**：跨文化使者的「去知识化」策略——猫不需要「知道」展品的历史/文化/宗教背景——这恰恰是观众的起点（大多数观众也不知道）。猫的「不知道」不是缺陷——是「叙事优势」——因为观众也是「不知道的」——猫的视角 = 观众的视角。如果换成专家旁白（「这件 16 世纪萨法维王朝的釉下彩瓷盘体现了…… 」）——观众立刻被推入「知识鸿沟」——「哦我不懂这个，这展览不是给我看的」。猫的「不懂」让展览变成「给所有人的」——「你不懂伊朗艺术？没关系——你看这只猫也不懂——但她觉得很好看——你也觉得很好看——这就够了。」
- **可迁移**：跨文化叙事的「共情载体」选择——当主题涉及陌生的文化/历史/地域时，选择一个「同样陌生」的主角（动物/儿童/外星人）——让主角的「陌生感」与观众的「陌生感」同步——主角的「发现之旅」= 观众的「发现之旅」。不选「专家」——专家已经「懂了」——观众无法「共情」一个已经知道答案的人。
- **适用场景**：studio-ad, custom

## 镜头语言（→ DP）

### 低角度动物跟拍：3cm 高度的卢浮宫

- **表现**：全片约 60% 的镜头在猫的眼睛高度（≈ 15-25cm，一只成年猫的肩高）。摄影机不是「放在地上仰拍」——而是「跟着猫移动」——猫走，摄影机走；猫跳上窗台，摄影机升到窗台高度；猫爬楼梯，摄影机斜向上跟拍。摄影机的高度/速度/方向被「绑定」在猫的身体上——观众不是在「看猫」——观众在「成为猫」。在卢浮宫宏伟的展厅中（天花板高约 10m），15cm 高的摄影机将空间重构为「巨物的森林」——卢浮宫从「庄严的人类文化殿堂」变成「一只猫的冒险游乐场」。
- **技术参数**：
  - 摄影机高度：15-25cm（猫眼高度），偶尔更低（贴地——模拟猫趴下时的视角 ≈ 5cm）
  - 运动方式：Steadicam/云台跟拍（稳定但不僵硬——有微小的「猫步节奏」——摄影机随猫的步伐轻微上下浮动——模拟「猫在走路时视角的呼吸感」）
  - 焦段：24-35mm 广角（广角在低角度下将空间「拉伸」——天花板看起来更高/走廊看起来更深——增强「巨物感」）
  - 构图：猫在画面中下位置（占画面 15-25%）——画面上方留给「建筑/展品/光影」——猫是「观察者」而非「被观察者」
- **可迁移**：动物跟拍的「猫步节奏」——摄影机不要完美平滑（轨道/dolly 的机械平滑 = 「人类观察者」而非「猫观察者」）。注入微小的高低浮动（±2-3cm，频率 ≈ 猫步频率 2-3Hz）——让观众潜意识感知这是「猫的视角」而非「摄影机视角」。实现方式：手持云台 + 步态同步（摄影师模仿猫的步伐节奏行走）+ 后期稳定（保留微小浮动，去除大幅晃动）。
- **适用场景**：custom, studio-ad

### 反射构图：水面 + 玻璃幕墙 = 空间的魔法门

- **表现**：全片最惊艳的转场都是通过「反射」实现的——不是后期数字转场——而是实拍中的反射。00:43 水面反射转场——白猫在卢浮宫的水池边——她「跳入」水中——准确说是她接近水面时画面切到水下视角——然后她的倒影「浮出」水面——但浮出的是浦东美术馆的水池——猫从巴黎的水面「潜入」、从上海的水面「浮出」。这不是「cross-dissolve」——这是「reflection cut」——水面 = 连通巴黎和上海的「魔法门」。00:56 玻璃幕墙转场——白猫在浦东美术馆室内看向玻璃幕墙——幕墙反射了蓝天白云和猫的影子——随着摄影机推近——反射变得越来越清晰——直到反射中的「猫」变成了真实的橘猫（另一只猫）——白猫和橘猫在「玻璃的镜像空间」中相遇。玻璃不再是「分隔内外」的屏障——它是「平行宇宙的连接面」。
- **技术参数**：
  - 水面转场：摄影机从俯拍水面→缓慢进入水下→「穿过」水面→浮出水面（摄影机从水面下方升出 = 视觉上「从倒影世界回到真实世界」——但实际上从巴黎倒影回到了上海真实）
  - 玻璃转场：白猫在玻璃内侧，橘猫在玻璃外侧——摄影机缓慢推近玻璃→玻璃的反射逐渐覆盖真实→直到我们看到的「白猫」其实是玻璃反射中的白猫 + 玻璃另一侧的橘猫——在同一平面相遇
  - 实现方式：水面转场 = 两次拍摄（巴黎水池+上海水池）+ 中间的水下过渡可能为 CG；玻璃转场 = 实拍（利用玻璃的反射特性——从特定角度拍摄可同时看到内侧的猫/外侧的猫/玻璃反射中的猫）
- **可迁移**：反射转场 ≠ 「省钱」（虽然确实比 CG 转场便宜）。反射转场的独特价值 = 「物理世界的魔法」——观众知道「水面反射」是真实的光学现象——所以「通过水面穿越空间」在视觉上是可信的（不需要「接受」CG 的虚构逻辑）。操作原则：(a) 选择反射面——水面/玻璃/镜面/金属/大理石地板——反射面越「日常」→ 转场越「神奇」（因为观众知道这个反射面是真实的——但他们看到的「穿越」是不真实的——这种「真实基础上的不真实」是反射转场的魔力来源）；(b) 反射转场需要双向拍摄——起点城市的反射画面 + 终点城市的反射画面——在后期中匹配反射的角度/光照/色温。
- **适用场景**：custom, studio-ad

### 瓷盘圆形构图：展品 = 画面的「瞳孔」

- **表现**：00:26 的瓷盘转场——白猫注视墙上的瓷盘——瓷盘逐渐放大——瓷盘的圆形轮廓变成画面的「虹膜」（iris）——画面从瓷盘的「瞳孔」中「走进」下一个场景（工作人员打包展品）。瓷盘的蓝色花纹在 iris 过渡中变成抽象的几何动画——最终「融化」为下一个镜头的蓝色调。这个转场完成了三件事：(a) 叙事——从「观赏展品」到「展品被运输」的空间跳跃；(b) 视觉——瓷盘的圆形 = 天然的 iris shape——不需要后期遮罩——只需要 scale up；(c) 符号——瓷盘上的图案（伊斯兰几何花纹）是展览的主题——这个转场让「图案」本身成为「空间之门」——图案不止是被看的——图案是活的——它可以打开通往另一世界的门。
- **技术参数**：
  - 实现方式：瓷盘实拍 → 数字 scale up + 圆形遮罩 → 瓷盘图案被提取为矢量几何动画 → 几何动画变形为下一场景的色彩底 → 底淡出揭示下一场景
  - 圆形遮罩的边缘处理：不是 hard edge（硬边 = 明显是遮罩）——而是 soft edge + 瓷盘边缘的自然阴影（让 iris 效果看起来像「相机镜头的光学暗角」而非「后期遮罩」）
  - 关键帧：瓷盘 scale up 过程持续约 1.5 秒——前 1 秒观众还能辨认「这是一个瓷盘」——最后 0.5 秒瓷盘填满整个画面→观众失去「瓷盘」的参照→进入「图案的抽象世界」→下一场景揭示
- **可迁移**：展品转场的「瞳孔公式」——选择具有「天然圆形轮廓」的展品（瓷盘/碗/钟表/镜头/戒指/穹顶/圆形画框）→ scale up 到充满画面 → 圆形成为天然 iris → 下一场景从 iris 中「睁开」。与随机形状的转场（直线 wipe/百叶窗）相比——圆形 iris 具有「生物性」（人眼瞳孔是圆的、相机光圈是圆的——圆形 iris 让观众潜意识觉得「这是一只眼睁开——我在看一个新世界」）。
- **适用场景**：product-demo, studio-ad

### 动态插画融合：实拍 + CG + 插画 = 三种「真实」的和平共处

- **表现**：全片混合了三种视觉媒介：(a) 实拍——猫/卢浮宫/浦东美术馆/展品；(b) CG——悬浮的瓷盘/飘动的织物/孔雀尾羽的绽放（这些不可能实拍的部分）；(c) 动态插画——旅程地图（00:35-00:42 的飞机飞越地图）、结尾信息展示（00:59-01:10 的展览信息动画）。三种媒介不是「分离的段落」——它们被编入同一个视觉流——实拍的猫在 CG 的悬浮瓷盘之间行走 → CG 的孔雀在实拍的展厅中开屏 → 飞机飞越的插画地图在猫的「想象」中展开 → 展览信息的插画在猫的「凝视」中浮现。三种媒介的边界被「猫的视角」缝合——因为猫是「真实」的（实拍），CG 和插画被「锚定」在猫的「主观世界」中——观众接受 CG/插画为「猫看到的/想象的」而非「后期加的特效」。
- **技术参数**：
  - 实拍层：猫 + 卢浮宫/浦东美术馆环境（真实光照/材质/物理）
  - CG 层：悬浮展品（瓷盘/壶/织物）+ 孔雀（需要突破物理限制的运动——展品不能真的悬浮——孔雀不能随意开屏）
  - 插画层：地图 + 展览信息（平面设计风格——伊斯兰图案+现代排版）
  - 三层融合的关键 = 光照统一——CG 层的打光必须匹配实拍层的光照方向/色温/软硬——否则 CG 层会「飘」在实拍层之上
- **可迁移**：三种媒介的「信任等级」——实拍 = 最高信任（观众知道这是真的），CG = 中等信任（观众知道这是「做的」——但好的 CG 可以骗过），插画 = 最低信任（观众明确知道这是「画的」）。融合策略：(a) 在实拍段建立信任（开头 0-15s 以实拍为主——让观众建立「这是真实的猫/真实的卢浮宫」的信任）；(b) CG 在实拍环境中「逐渐出现」——不是突然 cut 到全 CG 场景——而是实拍场景中逐渐加入 CG 元素（瓷盘开始悬浮——观众在「哦它在飞」之前已经接受了「瓷盘在这里」）；(c) 插画只在「想象/意识流」段落出现——当猫「凝视」某物时切入插画——观众自然将插画归入猫的「主观世界」。
- **适用场景**：custom, studio-ad

### 世界观/场景设计要点

- **「蓝金双色」的全片色彩霸权**：全片的色彩被严格控制在「蓝—金」双色轴上——几乎没有绿色/紫色/粉色。蓝 = 天空/水/钴釉/玻璃——「精神/永恒/跨越」，金 = 阳光/金箔/孔雀眼斑/木箱——「物质/珍贵/温暖」。蓝+金 = 伊斯兰艺术的终极色彩符号——全片用色彩本身做了「展览主题预告」——即使关掉旁白/文字/猫——只看色彩——观众已经「知道」这是关于伊斯兰/东方艺术的展览。
- **「白猫 = 白色的画笔」**：猫的纯白色不是偶然——在蓝金双色的世界里，白色猫 = 「移动的画布」——当猫走过深蓝展厅时 = 白色笔触在深蓝画布上「画画」；当猫站在金色织物前 = 白色主体被金色背景「框住」——类似波斯细密画中「白色主体+金色背景」的经典构图。猫的「白」在色彩功能上 = 画面中的「视觉锚点」——在丰富的蓝金图案中，观众的眼睛永远可以找到「白猫」——她是这个繁复世界中的「休止符」。
- **「反射 = 色彩的双重曝光」**：水面/玻璃幕墙的反射不仅是转场工具——它们是「色彩的双重曝光」。卢浮宫金字塔在水中的倒影 = 石灰白 + 水蓝的双重曝光（白的建筑被蓝的水「染色」）。浦东美术馆玻璃幕墙反射蓝天 = 白 + 蓝的双重曝光（白的建筑被蓝的天「染色」）。两种「白+蓝」的双重曝光在视觉上形成了巴黎和上海的「色彩对话」——两座城市的白建筑在同一片蓝天下——「虽然隔着 9000 公里——但它们在同一个色盘上。」



## 特效语言（→ VFX）

### CG 猫 + 实景融合：一只「不存在但真实」的猫

- **表现**：全片的猫看起来像实拍（毛发/运动/光影都很自然）——但考虑到猫在卢浮宫展厅中自由行走且展品悬浮/孔雀开屏等 CG 元素的交互——猫很可能是 CG 制作的（或部分 CG 增强）。CG 猫的挑战：(a) 白色毛发在深蓝背景下的边缘不能出现「CG 光晕」（白色边缘溢出 = 暴露 CG）；(b) 猫的运动必须有「猫的物理」（跳跃的抛物线/落地的缓冲/转身的重心转移——差 1 帧就不像猫）；(c) 猫与实拍环境的交互——猫脚踩在大理石地板上的微小滑动/爪子在织物上的钩挂——这些「不完美」是「真实感」的关键。如果猫是纯实拍——那也是极高水平的动物训练+拍摄调度——在卢浮宫真实展厅中让一只猫精准走位。
- **实现思路**：(a) 如果是 CG 猫——使用 Unreal Engine 或 Houdini 的毛发系统 + 动作捕捉（使用真实猫的 motion capture 或参考视频逐帧匹配）；(b) 光照匹配——实拍环境的光照用 HDRI 球采集→应用到 CG 猫的渲染中——确保 CG 猫的光照方向/色温/软硬与实拍环境一致；(c) 交互层——在猫需要「接触」实拍环境时（走在地板上/跳上窗台）——在实拍地板/窗台上放置物理标记点 → 后期用 CG 猫替换或叠加。
- **可迁移**：CG 动物的「不完美真实感」——CG 猫不能是「完美的」——完美 = 塑料感。真实猫的运动有：(a) 耳朵的微小抽动（即使身体静止——耳朵在持续扫描环境声音）；(b) 尾巴的随机摆动（猫的尾巴有自己的「意志」——不完全受身体运动影响）；(c) 毛发的「风动」——即使没有风——猫自己呼吸/运动产生的气流也会让毛发微动。这些「微动」是 CG 动物「活过来」的关键——每一帧都要有微小的随机运动——不能有「冻结帧」。
- **适用场景**：custom, studio-ad

### 图案→实物的「意识流转场」：瓷盘图案活过来

- **表现**：00:26 的瓷盘转场中——瓷盘上的蓝色几何花纹在 iris 放大过程中开始「动」——花纹的线条开始旋转/重组——像「活着的图案」——然后图案变形为下一个场景（工作人员打包展品）的色彩底。这不是「图案动画覆盖画面」——而是「图案从瓷盘上『逃走』——扩散到整个画面——然后『沉淀』为下一个场景」。中间的过渡（约 0.5 秒）是纯抽象的几何动画——观众短暂地「失去」了具象参照——进入了「图案本身的意识流」——然后被「拉回」到具象的下一个场景。这种「短暂失重」是转场的魔力——它割裂了叙事的时间线——但缝合了叙事的「情感线」——观众从「看展品」（外部观察）短暂进入「图案的意识」（内部体验）。
- **实现思路**：(a) 提取瓷盘的高清照片 → 将图案矢量化为 SVG 路径；(b) 在 After Effects 中对 SVG 路径做变形动画——花纹线条分离→旋转→重组为新的几何构图；(c) 变形动画的最后一帧 = 下一个场景的色彩构成（提取下一个场景的主色/辅色 → 在 AE 中用这些颜色填充几何构图 → 然后 fade 到下一个场景的实拍画面）；(d) 变形过程持续约 1-1.5s——节奏必须匹配音乐（在音乐的一个乐句内完成变形——在乐句结束时「落地」到下一个场景）。
- **可迁移**：图案转场的「来源-抽象-归宿」三步法：(a) 来源——从画面中提取一个「图案元素」（衣服花纹/建筑装饰/自然纹理/字体/logo）；(b) 抽象——将图案变形为纯几何动画（线条飞散/重组/旋转——0.5-1s 的「失重」期）；(c) 归宿——几何动画的色彩/构图「落地」到下一个场景。关键：来源的图案必须与归宿的场景有「色彩/形状的呼应」——否则转场变成了「两个不相关的东西被强行缝在一起」——观众会觉得卡。
- **适用场景**：studio-ad, logo-animation

### 水面反射空间切换：巴黎→上海的一「潜」之间

- **表现**：00:43 的水面反射转场是全片最核心的空间跳跃。白猫在卢浮宫水池边 → 她低头看水中的倒影 → 倒影中的白猫「向上看」（与现实中的猫形成对视）→ 摄影机「潜入」水中 → 穿过水面 →「浮出」水面——但浮出的是浦东美术馆的水池。整个过程约 2-3 秒。这不是「cross-dissolve from Paris to Shanghai」——而是「摄影机穿过水面 = 穿过空间的『膜』」。水面被表现为一个「半透膜」——一侧是巴黎、一侧是上海——猫和水面共同完成了这个「穿越」。
- **实现思路**：(a) 巴黎水池实拍——白猫在水池边，摄影机记录她的倒影；(b) 上海水池实拍——同样角度/光照/水面状态——摄影机从水下向上拍（模拟「浮出」的视角）；(c) 中间的水下过渡——CG 或实拍的水下画面（蓝色/模糊/气泡——让观众感知「我在水下」）；(d) 后期合成——巴黎水面倒影 → 0.5s CG 水下过渡 → 上海水面浮出；(e) 关键匹配点——两个水池的水面颜色/波纹状态必须接近——否则转场会「卡」。
- **可迁移**：水面转场的「半透膜」隐喻——水面不是「一面镜子」（纯反射）——也不是「一扇窗」（纯透明）——它是「半透膜」——同时反射+透射。利用这个物理特性——让水面成为「两个世界的交汇面」。操作：(a) 找到两个有「水面」的场景——水面颜色/状态越接近 → 转场越流畅；(b) 摄影机运动方向 = 「穿过水面」——不能只是从水面 A cut 到水面 B——必须让观众感知「我穿过了水面」——这要求中间有「水下」的过渡；(c) 水面上的「倒影主体」必须在两个场景中一致——巴黎水面倒影 = 白猫，上海水面也「浮出」白猫——主体的连续性 = 空间跳跃的可信性。
- **适用场景**：custom, studio-ad

## 声音设计（→ Sound Designer）

### 钢琴+弦乐优雅配乐：音乐 = 猫的脚步节奏

- **表现**：全片配乐是一首轻快的钢琴曲（配以少量弦乐和声）。钢琴的节奏不是「跟着画面走」——而是「钢琴的节奏 = 猫的脚步节奏」。猫在卢浮宫走廊中行走时的步频 ≈ 2-3 步/秒，钢琴的右手旋律以 16 分音符的节奏跑动——与猫的步频精确匹配。当猫停下来凝视瓷盘时——钢琴的左手和弦延长（sustain pedal），右手停止跑动——「音乐也在凝视」。当猫开始奔跑上楼梯——钢琴加速——音符密度增加。音乐不是「配乐」——它是「猫的情感脉搏」——猫的心情 = 音乐的 tempo——观众通过音乐感知猫的情绪——虽然猫没有面部表情。
- **声音层次**：钢琴+弦乐 (60%) > 环境音效 (25%) > 特殊音效 (15%) > 猫叫声 (0%) > 旁白 (0%)。钢琴被推到前景——不是因为「音乐很重要」——而是因为「猫没有声音」（没有对白/没有旁白/没有猫叫）——钢琴替代了猫的「声音」——钢琴 = 猫的内心独白。
- **可迁移**：动物主角的「音乐化声音」策略——如果动物主角没有「声音」（不叫/不说话），给动物分配一种乐器——猫 = 钢琴（优雅/独立/敏捷），狗 = 大提琴（忠诚/温暖/稳重），鸟 = 长笛/竖琴（轻盈/高远/自由）。乐器的「音色」= 动物的「性格」——让音乐成为动物的「画外音」——不需要动物「叫」——乐器替它「说」。
- **适用场景**：studio-ad, custom

### 盖章声 = 旅程序幕：一个声音开启一段旅程

- **表现**：00:27——工作人员在木箱上盖下「Louvre」印章——盖章声（「啪」——约 0.2s 的短促打击声——中低频 200-1000Hz + 高频 8000Hz 的清脆接触面）——这是全片第一次出现「非环境音」的明确音效。盖章声之前——观众沉浸在「猫在卢浮宫的梦幻旅程」中（环境音很少，主要是音乐）——盖章声突然「叫醒」观众——「旅程结束了——展览要出发了。」这个单一音效完成了叙事的「转折」——从「梦幻探索」切换到「现实运输」——从「巴黎的悠闲」切换到「旅程的紧张」——一个声音承担了叙事转折的全部重量。
- **声音层次**：盖章声的音量被推到 -3dBFS（接近峰值）——比钢琴音乐高出 6-8dB——故意的「声音惊吓」（startle effect）——让观众注意到「旅程开始了」。盖章声的混响被设置为极短（< 0.3s RT60——暗示「小空间/近距离」——与卢浮宫大厅的「大空间/远距离」形成对比——观众从卢浮宫的「广阔」被拉入打包间的「紧密」。
- **可迁移**：单音效叙事转折——在影片中段插入一个「与前段声音完全不同的」音效——声音的「断裂」= 叙事的「断裂」。前段 = 音乐为主 + 极少环境音（梦幻/沉浸），中段 = 一个突兀的单音效（现实/打断），后段 = 新的声音环境（运输/现实）。操作：(a) 转折音效的音量要比前段音乐高 6-10dB；(b) 转折音效的混响空间要与前段不同（前段 = 大空间/长混响，转折 = 小空间/短混响——用声学空间的对比强化叙事转折）；(c) 转折音效后立即切换到新的声音环境——不给观众「回到前段」的期待。
- **适用场景**：studio-ad, product-demo

### 无猫叫的克制：沉默的猫 = 庄严的展览

- **表现**：全片没有一声猫叫——虽然主角是猫。这是一个极度克制的选择——因为猫叫（meow）会立即触发观众的「可爱反应」（aww——好可爱）——而「可爱」不是本片想要的情感。本片想要的情感是「庄严/优雅/异域/奇妙」——猫叫会瞬间将情感从「庄严」拉到「可爱」——破坏全片的美学。替代方案：(a) 猫的脚步声音效——但不是「夸张的卡通猫步声」——是极轻微的「肉垫踩在大理石上的柔软声」（10-20dB，几乎可忽略——让观众潜意识感知但不会「听到」）；(b) 环境音——卢浮宫的微小声响（远处的脚步声/呼吸声/空调声——被压到极低音量——制造「空旷的庄严感」）；(c) 钢琴 = 猫的「声音」——猫不需要「叫」——钢琴替她表达了所有情感。
- **声音层次**：猫存在的唯一声音证据 = 极轻微的脚步声 + 毛发的摩擦声（在特写镜头时几乎不可闻的「沙沙」声——猫转头时毛发与空气的摩擦——这是拟音师在后期加的——观众不会「听到」但会「感觉到」猫的物理存在）。
- **可迁移**：动物主角的「去可爱化」声音策略——如果你的品牌/主题需要「庄严/优雅/高端」而非「可爱/搞笑/亲民」——让动物主角「沉默」。(a) 不给动物任何「可爱」的声音（叫/喘息/吃东西的声音）；(b) 用音乐/环境音替代动物的「声音表达」；(c) 动物的「存在」通过极轻微的身体音效传达（脚步声/毛发声/呼吸声——音量控制在 −25 到 −35dBFS——观众不会「听到」但会「感知到」。好的声音设计 = 观众说不出为什么觉得真实——但去掉这些声音后观众立即觉得「假的」。
- **适用场景**：studio-ad

## 创意策略（→ 广告片专属）

### 猫 = 文化大使：降低艺术展览的「门槛高度」

- **表现**：艺术展览宣传片的目标受众可以分成两类：(a) 「艺术爱好者」——已经知道卢浮宫/已经会去看展——不需要宣传片来说服；(b) 「潜在观众」——听说过卢浮宫但从未去过/对「艺术」有距离感「我不懂艺术」/可能被说服来看展。传统展览宣传片以「艺术爱好者」为目标受众（画面 = 展品特写+专家访谈——给「懂的人」看），本片以「潜在观众」为目标受众——用猫替代专家——猫不懂艺术——但猫享受艺术——这对「潜在观众」传达的信息是：「你不懂印度/伊朗/奥斯曼艺术？没关系——猫也不懂——但你们都觉得很美——来看吧。」
- **品牌目的**：浦东美术馆的目标不是「教育观众关于伊斯兰艺术的知识」——而是「让观众走进浦东美术馆」。猫 = 「走进来」的最佳代言人——因为猫不会教育你——猫只会邀请你——「跟我来——这里很美——不需要懂——只需要看」。
- **可迁移**：「去门槛化」的代言人选择——如果你的产品/服务有「知识门槛」（艺术/科技/金融/法律）——选一个「同样不懂但享受其中」的代言人（动物/儿童/外星人/第一次接触这个领域的人）——让代言人的「不懂」与目标受众的「不懂」对齐。不要选专家——专家会「教学」——教学 = 门槛——观众会觉得「我需要先学才能享受」。
- **适用场景**：studio-ad

### 双城记的普世情感：巴黎不是「高高在上」的圣地

- **表现**：通常的「世界级展览来华」宣传会强调「来自巴黎卢浮宫的珍贵展品——首次来到中国」——这种叙事的潜台词 = 「巴黎是艺术的圣地——中国观众幸运地能在『家门口』看到圣地来的珍品」。这种「朝圣叙事」隐含了文化等级（巴黎高于上海）——可能让部分中国观众不适（「又是欧洲中心主义」）。本片完全避开了朝圣叙事——卢浮宫和浦东美术馆被表现为「平等的对话」——猫可以从巴黎「跳」到上海只需要一个水面反射——两座城市的标志建筑在视觉上被平等对待（卢浮宫金字塔 = 浦东美术馆玻璃幕墙——都是「白+蓝」的建筑）——两座城市在猫的旅程中是「同一片蓝天下的两个美丽的家」——「展览来到上海」不是「施舍」——是「分享」。
- **品牌目的**：浦东美术馆需要建立自己的「国际地位」——而不仅仅是「卢浮宫的展览分站」。品牌信息：「我们是卢浮宫的平等合作伙伴——不是『下级单位』。」双城叙事的「平等性」直接传达了这一品牌信息——不需要说出来——画面已经说了。
- **可迁移**：跨文化品牌的「平等叙事」——如果你的品牌涉及两个文化/城市的合作——避免「从 X 到 Y」的单向叙事（X = 给予者，Y = 接受者）。用「X 和 Y 在 Z 中相遇」的双向叙事——Z = 一个「第三空间」（本片中的水面/玻璃反射）——在第三空间中，X 和 Y 是平等的。
- **适用场景**：studio-ad

## 适用提示

- **最佳匹配场景**：艺术/文化/博物馆/展览的宣传片；需要「跨文化/跨城市/双城」叙事的品牌内容；想用动物（尤其是猫）做「品牌代言人」的创意方向；需要「去门槛化/降低距离感」的高端品牌（艺术/奢侈品/酒店/旅游）；CG+实拍+插画三种媒介融合的混合媒体项目；需要「克制/优雅/不做可爱」的动物叙事。
- **不适用场景**：需要人类角色/对白/情感弧线的叙事（猫没有对白——全片没有人类角色——无法传达复杂的人际情感）；快节奏/高能量的广告（猫的节奏是慢的/优雅的——不适合运动品牌/能量饮料等品类）；需要直接销售转化（「立即购票」——本片是品牌形象片——不对直接转化负责）。
- **引用优先级**：核心引用（动物代言人叙事 / 双城蒙太奇 / 反射转场 / 瓷盘圆形 iris 转场 / CG+实拍+插画三层融合 / 无猫叫的克制声音设计 / 猫步节奏摄影机 / 蓝金双色色彩霸权）；补充参考（单音效叙事转折 / 展品即风景 / 去门槛化代言人 / 平等双城叙事）。
