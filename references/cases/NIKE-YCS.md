## 叙事技法（→ Writer）

### 隐喻剪辑：动作的普世语言

- **表现**：24 名运动员、36 种运动、53 组分屏匹配——没有一个镜头是「随机」的。每一组匹配都在说同一件事：虽然我们肤色/性别/运动/国家不同，但我们做着相同的动作、流着相同的汗、有着相同的决心。
- **手法**：核心隐喻 = 「不同的人，相同的动作 = 不同的人，相同的人性」。剪辑逻辑不是按运动类型分组（不是「所有篮球→所有足球」），而是按动作形态分组（跳 = 跳 / 挥拍 = 挥拍 / 拥抱 = 拥抱）。这让不同运动的运动员在同一个动作瞬间「相遇」。
- **可迁移**：任何需要强调「共性 > 差异」的叙事都可以用此技法。关键步骤：
  1. 梳理所有可用素材的动作类型（跳/跑/挥/踢/抱/跪/起）
  2. 在动作的「顶点」（跳跃最高点/挥拍击球瞬间/拥抱最紧时）做分屏匹配
  3. 匹配点的动作形态必须完全一致（偏差 > 5° 观众会感觉「不齐」）
- **适用场景**：product-demo, studio-ad

### 无旁白累积：用画面递增情绪

- **表现**：前 1 分 15 秒只有画面+音乐，没有一句旁白。画面从单体动作 → 双人分屏 → 四人分屏 → 多人群像——镜头的「人口密度」逐渐增加。Megan Rapinoe 的旁白在第 1:15 才进入：「When things seem impossible...」
- **手法**：情绪累积曲线 = 镜头数量 × 匹配精度。前半部分观众沉浸在「哇这个匹配太绝了」的视觉惊叹中，后半部分旁白进入时观众已经 emotionally primed，旁白的每一个词都带着前面累积的情绪重量。
- **可迁移**：如果旁白要承载品牌价值观信息（而非产品信息），先让画面累积 30-60s 的情感基础再进入旁白。旁白进入的最佳时机 = 观众已经「看懂了」但「还没说出来」的时刻。
- **适用场景**：product-demo, studio-ad

### 集体叙事：「我们」替代「我」

- **表现**：全片没有单一主角。24 名运动员没有一个被特别突出——所有人的镜头时长几乎均等（每人约 3-4 秒出镜）。品牌信息不是「Nike 让你成为冠军」，而是「我们（所有人）在一起就不可阻挡」。
- **手法**：用剪辑的平均主义构建「集体」身份。关键约束：
  - 任何运动员出镜 ≤ 2 次（避免明星效应）
  - 性别比约 50/50
  - 运动类型覆盖：主流运动（篮球/足球/网球）+ 小众运动（滑板/攀岩/轮椅篮球）
- **可迁移**：需要传达「品牌属于所有人」时，严格控制个体出镜时长。设置硬约束：任何个体 ≤ 总时长的 5%。
- **适用场景**：product-demo, studio-ad

## 镜头语言（→ DP）

### 分屏匹配的几何精度

- **表现**：中线是可见的黑色细线——它不是被隐藏的。56 组分屏匹配中，运动员的动作在中线两侧完美对称/互补。一名篮球运动员的左臂与一名足球运动员的右腿在分屏中线处「相接」——看起来像同一个身体的延伸。
- **技术参数**：
  - 分屏比例：固定 50/50 垂直分割（偶有三联屏）
  - 匹配点：动作顶点（跳跃最高点/击球瞬间/转身 180° 时）
  - 构图约束：每个被匹配的运动员在各自半屏中的位置必须镜像对称。如果有人偏左 → 对侧的人必须偏右。
  - 缩放匹配：不同素材的焦距/画幅不同，通过数字缩放（最高放大 120%）统一构图。
- **可迁移**：做分屏匹配的步骤：
  1. 建立素材库 → 标注每个镜头的「动作类型 + 动作方向 + 摄影机角度」
  2. 匹配时优先选择相同动作类型 + 相反动作方向的素材（A 向左 → B 向右）
  3. 分屏线置于画面的几何兴趣点（0.382/0.618 黄金分割），不一定是正中间
- **适用场景**：product-demo, logo-animation

### 档案素材的统一化处理

- **表现**：4,000+ 小时素材来自不同年代（1990s-2020）、不同拍摄条件（专业体育转播/手机拍摄/监控录像）。Oscar Hudson 团队通过统一的后期处理让所有素材看起来属于「同一个世界」。
- **技术参数**：
  - 色彩统一：所有素材套用相同的 LUT（低饱和度 + 暖底色 + 暗部提灰），但保留素材原有肤色
  - 画幅统一：所有素材裁切到 1.85:1（接近 16:9）
  - 颗粒统一：较新的数字素材加轻微 16mm 胶片颗粒（模拟），使它们与老旧档案素材在质感上融合
  - 运动补偿：摇晃的素材做稳定化处理（但留有 5-10% 的原生抖动以保持真实感）
- **可迁移**：处理多来源素材的统一化公式：
  1. 先统一画幅（裁切到目标宽高比）
  2. 再统一色调（LUT + 暗部提灰）
  3. 再加统一质感（颗粒/锐度）
  4. 最后做运动稳定（适度保留原生抖动）
  顺序不能乱——先调色再稳定 vs 先稳定再调色的结果不同。
- **适用场景**：product-demo

### 动态节奏的数学设计

- **表现**：剪辑节奏不是均匀的——它从低速（约 3s/镜头）加速到高速（约 0.8s/镜头），然后在旁白进入时降到低速（2s/镜头），再次加速至高潮（0.5s/镜），最终在 Slogan 时完全停止（3 秒黑屏+Just Do It）。
- **技术参数**（按时间轴）：
  - 0:00-0:20：平均 2.5s/镜头（建立视觉语言）
  - 0:20-0:50：加速到 1.0s/镜头（分屏匹配密集段）
  - 0:50-1:15：变速 0.5-2.0s/镜头（音乐节奏变化段）
  - 1:15-1:25：降到 2.0s/镜头（旁白进入，给观众「呼吸」）
  - 1:25-1:30：加速到 0.4s/镜头 → 静止（Slogan 落地）
- **可迁移**：设计剪辑节奏曲线——不是均匀的，而是有「呼吸感」的。高潮前必有缓坡（让观众准备），爆发后必有静止（让信息落地）。参数化方案：`节奏曲线 = f(时间) = a×sin(b×时间) + c×时间`（a 控制波动幅度，b 控制波动频率，c 控制整体加速趋势）。
- **适用场景**：product-demo, logo-animation

## 色彩/美术（→ Art Director）

| 场景/段落 | 主色 (hex) | 辅色 (hex) | 情绪 | 设计理由 |
|-----------|-----------|-----------|------|---------|
| 全片统一底色 | `#1A1A1A` 近黑 | `#2D2D2D` 暗灰 | 史诗、严肃 | 深色背景让运动员的身体和运动服颜色「跳出来」。分屏中线也是黑色，融入背景。 |
| 肤色暖调处理 | `#D4A574` 暖肤 | `#8B6914` 深肤 | 人性、温暖 | 肤色是画面中唯一的「暖色区」。不管原始素材的色温如何，肤色被校正到同一温区。 |
| 运动服亮色 | `#FF6B35` Nike橙 | `#00A86B` 绿 | 能量、品牌 | Nike 的标志性橙色在多个镜头中作为服装/鞋子/配件出现——不喧宾夺主但始终存在。 |
| 黑白档案素材 | `#B0B0B0` 中灰 | `#E0E0E0` 亮灰 | 历史、尊重 | 黑白素材保留原貌（不着色），与彩色素材并置 = 「过去和现在同在一个画面」。 |
| Slogan 黑屏 | `#000000` 纯黑 | `#FFFFFF` 纯白 Swoosh | 权威、终结 | 90 秒的能量累积后，纯黑+白 Swoosh 像一堵墙——「不要再多说一个字，这就是答案」。 |

### 世界观/场景设计要点

- **「运动场即世界」**：每个运动场地（篮球场/足球场/跑道/游泳池/街道）被当作一个「国家」，分屏线就是国界线——但运动员的动作跨越了界线。
- **服装的色彩策略**：不统一服装颜色，让多样性成为调色板的一部分。但通过后期降低饱和、统一暗部来避免色块冲突。
- **品牌色克制使用**：Nike Swoosh 和品牌橙色仅在约 5% 的画面中出现。大部分时间观众看到的是「运动员」而非「Nike 产品」。但在最后 3 秒 Slogan 时品牌色全力释放。

## 特效语言（→ VFX）

### 分屏匹配的隐藏艺术

- **表现**：56 组分屏匹配中，部分匹配使用了数字合成来增强匹配精度。例如一名体操运动员的旋转与一名滑板运动员的旋转——背景被替换以统一视觉，但主体的运动完全保留。
- **实现思路**：
  1. 选出匹配对 → 分析动作顶点的时间差（通常在 1-3 帧内）
  2. 微调其中一段的速度（±5%）使动作顶点精确对齐
  3. 可能需要 roto 提取主体 + 替换背景（约 20% 的匹配需要）
  4. 分屏线加 1-2px 黑色分隔线（故意暴露分屏，制造「两个世界在交界处相遇」的感觉）
- **可迁移**：
  - HyperFrames：用 CSS Grid `grid-template-columns: 1fr 1fr` + 两个独立的 `<video>` 元素 → 精确控制每个视频的 `currentTime` 实现帧对齐。
  - ComfyUI：对于静态分屏画面，用 `LatentComposite` 或 `ImageCompositeMasked` 做精确拼接。提示词中加入 `split screen, two athletes, matching pose, center composition`。
- **适用场景**：product-demo

### 动态节奏变速

- **表现**：部分匹配镜头在动作顶点处做极短暂的慢动作（约 0.2-0.3s 的 50% 速度），让观众在那一瞬间「看清楚」匹配的精度，然后立刻恢复原速。
- **实现思路**：时间重映射（time remapping）——在动作顶点前后 0.5s 区间做速度曲线（ease-in → 50% → ease-out），其余部分 100% 速度。
- **可迁移**：
  - HyperFrames：用 JavaScript 控制 `<video>.playbackRate`——在关键帧位置将 rate 从 1.0 平滑过渡到 0.5 再回到 1.0。
  - 编辑软件：Premiere Pro 的 Time Remapping 关键帧 / DaVinci Resolve 的 Retime Controls。
- **适用场景**：product-demo, logo-animation

## 声音设计（→ Sound Designer）

### 动态节奏的声画对位

- **表现**：音乐节奏（鼓点）与剪辑节奏不是精确 1:1 对应——而是在「接近但不完全」的关系中制造张力。鼓点落在动作顶点「之后」约 0.1-0.2 秒 → 观众的大脑会主动「补全」这个微小的时间差，从而更投入。
- **声音层次**：音乐 (60%) > 现场环境声 (25%) > 音效 (10%) > 旁白 (5%)。现场环境声（球撞击声/呼吸声/鞋摩擦声）被保留并放大——制造「你真的在现场」的临场感。
- **可迁移**：声画对位不要追求完美同步。故意偏移 0.1-0.2 秒（声音略微落后于画面）可以制造「音乐在回应画面」的感觉，比精确同步更有生命力。`BPM = 110`（Drumless hip-hop 节拍 + 弦乐 Padding）。
- **适用场景**：product-demo, studio-ad

### 静默爆发的力量

- **表现**：全片最响的时刻是在 Slogan 出现之前——音乐在最后 1 秒突然收住，0.5 秒的完全静默，然后「Just Do It」出现在纯黑屏幕上。0.5 秒的静默比任何音乐都响。
- **声音层次**：音乐戛然而止 → 绝对静默 (0.5s) → Slogan 无声字幕。此时画面上的「Just Do It」自带声音——每个观众在脑海中用自己的声音念出这句话。
- **可迁移**：如果你的 Slogan/品牌名足够强（观众已经知道它），考虑用「静默 + 纯画面」来落地。声音的缺席 = 最大的存在感。时机参数：静默时长 ≤ 1s（超过 1s 观众会以为播放器坏了）。
- **适用场景**：product-demo, logo-animation

### 旁白的时机与重量

- **表现**：Megan Rapinoe 的旁白只有 3 句话——分别在 1:15 / 1:22 / 1:28 进入。每句话之间间隔 5-7 秒，给画面充分的「回应时间」。播放声音经过轻微压缩 + 近距离麦克风效果（ASMR 般的亲密感）。
- **声音层次**：旁白音量 = 音乐音量的 85%——不是压过音乐，而是「漂浮在音乐之上」。EQ 处理：中频提升 +3dB（200-800Hz，人声核心频段），音乐在这个频段做 -2dB 凹陷。
- **可迁移**：旁白不要「压过音乐」——在 EQ 上给旁白留出频段空间（音乐在旁白频段做 -2dB 凹陷），音量略低于音乐 10-15%。压制感 > 喧宾夺主。
- **适用场景**：product-demo, studio-ad

## 创意策略（→ 广告片专属）

### 品牌精神替代产品展示

- **表现**：全片 90 秒没有展示任何 Nike 产品的特写或功能。所有产品（鞋子/衣服/装备）都「在使用中」——它们不被打断、不被凝视。Nike 不是「卖运动装备的公司」——它是「推动人类突破极限的力量」。
- **品牌目的**：在 2020 年全球疫情 + 社会运动的背景下——传达「你们分不开我们，你们阻止不了我们」。品牌不是用口号说这句话，而是用 24 名运动员的动作证明了它。
- **可迁移**：如果你的品牌有强烈的价值观（而非独特的产品功能），考虑完全放弃产品展示。让品牌成为「价值观的载体」而非「产品的标签」。风险：如果价值观和品牌历史关联不够深，容易显得空洞。Nike 能这样做是因为 30 年「Just Do It」的品牌资产。
- **适用场景**：product-demo, studio-ad

### 集体共鸣的剪辑设计

- **表现**：广告中最动人的瞬间不是技术最复杂的匹配——而是一个戴头巾的女穆斯林击剑运动员与一名男篮球运动员在同一帧中同时「起跳」。观众在看到这个瞬间时不会想「这两个运动有什么关系」——他们会想「我们是一样的」。
- **品牌目的**：制造一个「每个人都看到自己」的画面——不同年龄/性别/种族/信仰/运动的观众都能在广告中找到「自己的那一个运动员」。
- **可迁移**：做「集体共鸣」型广告时，覆盖广泛的人群类型比追求视觉完美更重要。如果资源的 80% 能覆盖 80% 的目标人群，优先选择「覆盖」。
- **适用场景**：product-demo, studio-ad

### Slogan 落地的蓄力结构

- **表现**：全片 90 秒的视觉和情绪能量全部蓄到最后一帧——Swoosh + Just Do It。前面的一切（53 个匹配/24 个运动员/4000 小时素材）都是为这 3 秒准备的。这是典型的「蓄力 → 爆发」广告结构。
- **品牌目的**：不让 Slogan 只是一句口号——而是前面 87 秒累积的证据迫使观众在最后 3 秒得出的「唯一结论」。观众不是被告诉「Just Do It」——他们是自己「想」到这句话的。
- **可迁移**：Slogan 出现前的 5 秒是「证据的最后一击」——必须是全片最浓缩、最有力的画面。Slogan 之后 0 内容（不继续展示产品/明星/网站）——让 Slogan 成为句号。
- **适用场景**：product-demo, logo-animation

## 适用提示

- **最佳匹配场景**：需要传达品牌精神/价值观的广告、运动/竞技类视频、需要大量素材整合的创作、分屏/对比叙事结构。
- **不适用场景**：需要详细产品特写的功能演示、个人化叙事（单一主角）、幽默/轻松调性的内容。
- **引用优先级**：核心引用（分屏匹配 / 动态节奏 / 档案素材处理 / 集体叙事 / 品牌精神传递）。
