## 叙事技法（→ Writer）

### 极简信息设计：只有 3 个信息点

- **表现**：全片 1 分 16 秒，只有 3 个信息传递给观众：(1) 两台 Volvo 卡车在精确倒车； (2) Jean-Claude Van Damme 站在后视镜上做劈叉； (3) 「这个测试是为了展示 Volvo 动态转向系统的稳定性与精确性」。没有铺垫、没有冲突、没有解决、没有产品功能列表、没有价格标签。叙事 = 产品演示本身。
- **手法**：信息密度的「3 点极限」——人类工作记忆（working memory）在被动观看广告时只能同时处理 3-4 个独立信息单元。超过 3 个 = 观众开始遗忘前面的信息。Volvo Split 的 3 点设计：(a) 画面信息（倒车+劈叉——视觉输入）；(b) 旁白信息（Van Damme 的人生哲学——听觉输入）；(c) 字幕信息（"This test was set up to demonstrate..."——确认性输入）。3 个通道互不竞争（视觉/听觉/文字各占一个通道）。
- **可迁移**：广告信息设计的 3 点原则：(a) 每个信息点占一个感官通道（视觉/听觉/文字——不要 2 个信息竞争同一个通道）；(b) 信息点总数 ≤ 3（第 4 个会挤掉前 3 个中的 1 个——你的核心信息可能被挤掉）；(c) 最重要的信息放在最后（观众记住最后听到/看到的）。
- **适用场景**：product-demo, studio-ad

### 产品演示即叙事

- **表现**：这支广告没有「故事」——但「一个人在两台倒行的卡车间劈叉」本身就是完整叙事。观众的叙事体验：(a) 看到倒车的卡车和站在后视镜上的人 ——「他要干什么？」(b) 他张开双臂 ——「他不会要……？」(c) 他劈叉 ——「天哪他真的做了」(d) 字幕解释 ——「原来这是为了展示转向精度」。没有「从前有一个人……」——但有完整的起承转合。
- **手法**：产品演示的「叙事化」——不是「展示功能 → 解释功能」，而是「设置悬念 → 极致表演 → 揭示目的」。产品演示被包裹在「人会掉下去吗？」的叙事张力中——观众不是在看产品演示，而是在看一场极限表演。产品功能（转向精度）不再是被「说明」的——它是被「证明」的。
- **可迁移**：产品演示叙事化的 4 步模型：(a) 设置悬念——「这个人在干什么？会不会掉下去？」(b) 升级张力——「卡车继续倒行，他张开双臂」(c) 高潮/释放——「他完成了劈叉，卡车继续平稳倒行」(d) 揭示——「这是为了证明我们的转向精度」。每一步都是视觉的（不需要旁白解释悬念），产品在最后一步才「现身」。
- **适用场景**：product-demo

### 人生哲学旁白：与产品无关的深度

- **表现**：Van Damme 的旁白内容与卡车完全无关——他在讲自己的人生哲学：「我经历过很多艰难时刻……我的身体经过多年的武术训练……真正的挑战不是身体的——是精神的……当你在空中飞行，最困难的是保持平衡」。旁白在讲「人生/坚持/平衡」，视觉在展示「卡车倒行/人体劈叉」。两者在「精确/控制/平衡」这个主题上汇聚，但从未直接连接。
- **手法**：「平行叙事 + 主题汇聚」——不是「产品 A 好因为 B」，而是「主题 T（平衡/控制/信任）既存在于 Van Damme 的人生哲学中，也存在于 Volvo 的工程哲学中——两者在结尾由观众自己完成连接」。观众感觉「我自己发现了这个关联」而不是「广告告诉我这个关联」——自我发现的信息留存率是被动接收的 5 倍以上。
- **可迁移**：人生哲学旁白的适用条件：(a) 旁白者的「人生经历」必须与产品的「核心价值」有非直接的共鸣（Van Damme = 武术/平衡/控制 vs Volvo = 精确转向/稳定）；(b) 旁白不提及产品——产品只在最后字幕出现（如果旁白提了产品，就变成「名人推荐」而非「哲学共鸣」）；(c) 旁白的语言必须「文学化」而非「广告化」——像在 TED 演讲，不像在读提词器。
- **适用场景**：product-demo

## 镜头语言（→ DP）

### 一镜到底航拍：固定机位的震撼

- **表现**：全片是一个连续的单镜头——无人机航拍固定在中远景，摄影机不切、不推、不拉、不摇。Van Damme 在两台卡车正中间做劈叉，身后是晨曦中的地平线。没有「特写/中景/全景」的剪辑变化——摄影机选择「不干涉」，把所有的注意力留给「正在发生的事」。
- **技术参数**：
  - 机位：无人机航拍，离地约 20-30m，镜头朝正前方
  - 焦段：约 50-85mm（中长焦）——压缩远景，让地平线看起来更近，增强「史诗感」
  - 构图：完美对称——Van Damme 在画面正中央（x = 50%），地平线在画面 y = 55%（略低于中心——给天空留更多空间 = 开阔感）
  - 拍摄时间：黎明（日出前 20-30 分钟到日出后 10 分钟）——Golden Hour 的低角度阳光从侧面打来
  - 公路：笔直的高速公路（西班牙某段封闭公路）——无限延伸的透视消失线 = 旅程/永恒
- **可迁移**：一镜到底航拍的史诗感配方：(a) 对称构图（主体在画面中心——偏离中心 = 失衡 = 不适合「精确/控制」主题）；(b) 长焦压缩（50-85mm 压缩背景，让地平线「靠近」——广角会让主体渺小）；(c) 固定机位（摄影机不动 = 让运动发生在画面内部——摄影机运动 = 观众的注意力被「摄影」本身吸引）；(d) 选择黎明拍摄（低角度暖光 + 长阴影 = 天然的史诗光）。
- **适用场景**：product-demo

### 对称构图的极致运用

- **表现**：Van Damme 在两台卡车的正中间——左右对称（两台卡车镜像般平行倒行）。他的身体在中轴线上——伸开的双臂、水平的双腿——形成一个完美的「十」字。地平线在他身后，天空占了画面的上半部分。所有视觉元素都在强化「精确/平衡/控制」。
- **技术参数**：
  - 左右对称：两台卡车在画面左右各占约 20%，间距完全相等——卡车后视镜的位置误差 < 5cm（需要精确的驾驶员配合）
  - 中轴对齐：Van Damme 的身体中心线 = 画面中轴线，偏差 < 1%
  - 黄金比例：Van Damme 头部 ≈ 画面 y = 40%（黄金分割点）——最重要的视觉信息（面部）在最舒适的阅读位置
  - 水平线：地平线 = 画面 y = 55%——略低于中心 = 天空（自由/可能）占 55% > 地面（现实/重力）占 45%
- **可迁移**：对称构图的适用主题：(a) 力量/权威（T 字对称）——如 Volvo Split；(b) 平衡/和谐（镜像对称）——如镜像水面反射；(c) 精确/工程（几何对称）——如产品特写的轴线对齐。不适合：运动/混乱/自然——这些主题需要不对称构图来传递「活力」。
- **适用场景**：product-demo

### 黎明金色时刻的光线运用

- **表现**：从 Van Damme 身后打来的阳光是低角度的晨曦——在他身上形成轮廓光（rim light），让他的身体从暗色背景中「切割」出来。同时逆光让他的面部处于柔和的阴影中（不是死黑——天空的反射提供了柔和的 fill light）。
- **技术参数**：
  - 光源：日出前后 15 分钟的太阳——色温约 3000-3500K（warm golden），角度约 5-10° 仰角
  - 逆光 + 天空 fill：面部曝光由天空反射光提供（约比直射光低 2-3 档）——保留面部细节但保持轮廓的戏剧性
  - 卡车金属反光：晨曦在卡车银色表面形成长条形的金色反射——「工业的冰冷 + 黎明的温暖」= 反差美学
  - 长阴影：太阳低角度 = Van Damme 在路面上的影子被拉长到约 15-20m——影子 = 「人的尺度被放大」
- **可迁移**：Golden Hour 拍摄的广告规则：(a) 如果主体需要轮廓光（史诗/英雄感）→ 逆光拍摄（太阳在主体身后）；(b) 如果主体需要面部细节（温暖/亲切感）→ 侧光或正面光（太阳在侧面）；(c) Golden Hour 的有效时间窗口约 15-20 分钟（太阳角度变化快——复杂镜头需要多天在同一时间重复拍摄）。
- **适用场景**：product-demo

## 色彩/美术（→ Art Director）

| 场景/段落 | 主色 (hex) | 辅色 (hex) | 情绪 | 设计理由 |
|-----------|-----------|-----------|------|---------|
| 晨曦天空 | `#FFB74D` 暖金 | `#E3F2FD` 淡蓝 | 史诗、希望、起点 | 暖金 = 黎明的光线（"新的开始"）。淡蓝 = 清晨天空的冷底色（冷暖共存 = 一天的开始）。 |
| 灰色公路 | `#757575` 中灰 | `#BDBDBD` 浅灰 | 精确、工业、理性 | 灰色 = 公路的「中性舞台」——灰调不抢占注意力，让金色光线和银色卡车成为主角。 |
| 卡车金属银 | `#CFD8DC` 银灰 | `#90A4AE` 暗银 | 工程、可靠、冷峻 | 银色 = 工业精度。晨光在银面上的金色反射（非纯银 = 暖光的影响）。 |
| Van Damme 服装 | `#37474F` 深蓝灰 | `#263238` 近黑 | 专注、力量、克制 | 深色服装 = 在金色背景中形成对比 = 「人是画面中最暗的点 = 视觉焦点」。 |
| 结尾 Volvo 标识 | `#1565C0` Volvo 蓝 | `#FFFFFF` 白色文字 | 品牌、信任、确认 | 蓝色 = Volvo 品牌色，在 1 分钟的金色/灰色后第一次出现——蓝/金的互补色关系让品牌色冲击力最大化。 |

### 世界观/场景设计要点

- **三色极简主义**：全片的色彩只有 3 种——金色（晨曦）、灰色（公路）、银色（卡车）。没有第 4 种颜色。这种「色彩纪律」是刻意为之——每种颜色承担一个叙事功能：金色 = 情感/史诗，灰色 = 现实/工业，银色 = 产品/工程。如果加入了第 4 种颜色，任何一个颜色的叙事功能就会被稀释。
- **「公路 as 舞台」**：公路被当作舞台来设计——笔直、无限延伸、两侧无任何视觉干扰（没有路牌/护栏/建筑）。清理视觉噪音的极致——让观众的注意力 100% 在卡车上。
- **服装的「反英雄」设计**：Van Damme 穿着深色牛仔裤和深色夹克——不是超级英雄的战服，也不是运动员的紧身衣。这是「真实的人在真实地做一件超凡的事」的服装语言——观众更容易代入（「如果是我……」）。
- **卡车的「不美化」原则**：FM 卡车在广告中保持了工业原貌——没有 custom paint、没有 chrome 装饰、没有洗得发亮。Volvo 不需要让卡车「看起来漂亮」——它需要让卡车「看起来可靠」。工业美学 > 消费美学。

## 特效语言（→ VFX）

### 安全绳擦除：唯一的 VFX

- **表现**：Van Damme 真的站在两台倒行卡车的后视镜上做劈叉——但他身上有安全绳（如果卡车偏了，他不会真的掉下去）。后期唯一的工作：擦除安全绳。这是「诚实 VFX」的典范——VFX 的存在是为了消除 safety measure 的痕迹，不是为了让不可能变成可能。
- **实现思路**：(a) 安全绳从 Van Damme 的 harness 连接到两台卡车的车身（确保即使他失去平衡也不会坠落）；(b) 安全绳选细钢缆（直径 3-4mm），颜色选接近天空的浅灰（减少后期擦除的难度和残留痕迹）；(c) 后期用 clean plate（无 Van Damme 无安全绳的纯背景 shot）+ 逐帧 rotoscope 擦除安全绳；(d) 天空背景的渐变让擦除痕迹很难被察觉（没有硬边缘/复杂纹理）。
- **可迁移**：安全装置隐匿的原则：(a) 安全装置颜色选接近背景色（天空背景用浅灰/浅蓝，地面背景用深灰）；(b) 安全装置选最细但足够的直径（钢缆 > 绳索——钢缆更细 + 无绒毛边缘 = 更容易擦除）；(c) 背景越简单（天空/纯色/渐变）→ 擦除越不可见。背景越复杂（树叶/人群/建筑纹理）→ 擦除越难。
- **适用场景**：product-demo

### 零 CGI：实拍 stunt 的不可替代性

- **表现**：Van Damme 真的在两台倒行的卡车上做了劈叉。观众知道这一点——因为如果他没做，CGI 版本的「恐惧感」会完全不同。CGI 可以无限接近真实，但观众潜意识知道「这是特效」——真实人体在真实危险中的 tension 是无法被 CGI 复制的。
- **实现思路**：(a) Van Damme 本人完成（非替身——53 岁的他坚持自己完成）；(b) 两台卡车由专业驾驶员在封闭公路上倒行，速度约 15-20 km/h；(c) 后视镜上安装了定制的平台（刚好够 Van Damme 的双脚站立——约 15cm × 30cm）；(d) 拍摄多次（实际拍摄次数未公开），每次都需要完美的同步——两台卡车的速度/方向/间距必须精确一致。
- **可迁移**：实拍 stunt vs CGI 的决策原则：(a) 如果核心信息是「产品的精确/可靠」→ 实拍（真实 = 信任）；(b) 如果核心信息是「想象力的极限」→ CGI（超现实 = 想象力）；(c) 如果核心信息是「人的极限」→ 实拍（CGI 的人体极限不是「极限」——因为没有风险）。Volvo Split 选择 (a)+(c)——产品精确 + 人体极限 = 必须实拍。
- **适用场景**：product-demo

## 声音设计（→ Sound Designer）

### Enya "Only Time" 的音乐反差

- **表现**：Enya 的 "Only Time"——柔美女声 + 空灵编曲 + 多层次合成音效——被配在两台重型卡车和一个肌肉男的极限 stunt 上。这是音乐反差的第二经典案例（第一是 Sony Balls 的 "Heartbeats"）。Enya 的空灵 = 让卡车的「工业感」变得「神圣」。
- **声音层次**：音乐 (50%) > 旁白 Van Damme (35%) > 环境音/卡车引擎声 (15%)。音乐不是「背景」——它是旁白的「counterpoint」——Enya 在唱「谁能说你的心去了哪里 / 只有时间」，Van Damme 在说「我经历过很多艰难」，两者的主题在「时间/坚持」上汇聚。
- **可迁移**：音乐反差的进阶用法：(a) 柔美女声 + 硬核工业 = 将工业「神圣化」（Enya × Volvo）；(b) 安静木吉他 + 视觉爆炸 = 让爆炸「内省化」（Heartbeats × Sony Balls）；(c) 古典交响乐 + 体育动作 = 让运动「史诗化」（Nike 常用）。选择反差音乐的公式：音乐的「情感类型」和视觉的「物理属性」成反比。
- **适用场景**：product-demo

### 信息层的听觉分离

- **表现**：全片的声音包含 3 个独立的信息层——但它们在听觉上被精心分离，不会互相干扰：(a) 音乐层（Enya——贯穿全片，音量恒定）；(b) 旁白层（Van Damme——从 0:15 进入，音量略低于音乐峰值 3dB）；(c) 环境层（卡车引擎——仅在开头和结尾清晰，中间被音乐+旁白遮蔽）。3 个层的频段不同——Enya 占据中高频（200-4000Hz），Van Damme 的男中音占据中低频（150-500Hz），卡车引擎占据低频（50-150Hz）——频谱不重叠。
- **声音层次**：频谱分离策略——音乐的中高频人声占比 70%，为 Van Damme 的中低频预留空间（通过 EQ: Enya 在 150-500Hz 衰减 3dB，Van Damme 在 150-500Hz 增强 2dB）。环境音（卡车引擎 50-150Hz）在旁白开始后压缩 -6dB（side-chain: 旁白触发 → 引擎音量衰减）。
- **可迁移**：多信息层音频的频谱分离公式：(a) 对话/旁白频段（150-500Hz 基频 + 2000-4000Hz 清晰度）——其他音轨在这个频段衰减 3-6dB；(b) 音乐频段（全频段，但避开人声频段——通过 dynamic EQ 实现）；(c) SFX/环境频段（低频 < 150Hz + 高频 > 6000Hz——这些频段不与人声竞争）。操作工具：Trackspacer / Soothe2（动态频率避让插件）。
- **适用场景**：product-demo, studio-ad

### Van Damme 的旁白录音处理

- **表现**：Van Damme 的旁白听起来像他在对着一个人的耳朵说话——而不是在朗读广告词。声音的「亲密感」来自：(a) 近距离录音（话筒距离 15-20cm——不是录音室标准的 30-40cm）；(b) 保留气息声（不切除低频气息——这是「真人」的听觉签名）；(c) 轻微的比利时口音不修正（口音 = 真实性——完美的播音腔 = 广告感）。
- **声音层次**：近距录音 + 轻压缩（2:1 ratio, -8dB threshold）+ 微量混响（RT60 ≈ 0.4s, small room preset——不是 dry voice，需要一点空间感但不要「录音室感」）+ 保留 80-120Hz 的低频（胸声共鸣——这是「权威感」的来源，但广告录音通常会切掉这个频段）。
- **可迁移**：旁白录音的「去广告化」配方：(a) 话筒距离缩短到 15-20cm（近距离 = 亲密感/ASMR 效果）；(b) 压缩比降到 2:1（重压缩 = 广告/广播感）；(c) 保留口音/气息/非标准发音（这些 = 「真实的人」——消除它们 = 「广告播音员」）；(d) 混响用「小房间」而非「录音室吸音」（录音室太干净 = 没有人情味）。
- **适用场景**：product-demo

## 创意策略（→ 广告片专属）

### 产品演示即表演：极限信任证明

- **表现**：「如果我们的转向系统不够精确，我就掉下去了。」——这不是台词（全片没有这句），但观众在 Van Damme 站上去的那一刻就已经在想这句话了。产品功能（动态转向精度）没有被「说明」——它被一个 53 岁的动作明星用自己的身体「证明」了。这是「信任证明」的最高形态——不是「我们测试过」（实验室数据），不是「客户说好」（证言），而是「我拿命来证明」。
- **品牌目的**：Volvo Trucks 的核心品牌资产 = 「安全」。但卡车司机不读安全测试报告——他们相信自己看到的。Van Damme 在两台卡车间劈叉的画面比任何碰撞测试数据都有说服力——因为「一个人的命」比「测试假人的数据」更有情感重量。
- **可迁移**：极限信任证明的 3 级递进：(a) Level 1: 数据证明（「测试结果精确到 0.01mm」）——最弱；(b) Level 2: 专家证言（「工程师说它很精确」）——中等；(c) Level 3: 生命证明（「如果不够精确，我就掉下去」）——最强。要跃升到 Level 3，需要找到一种「如果产品失败 = 严重后果」的表演形式——后果越严重（受伤/死亡/毁灭），证明越有力。
- **适用场景**：product-demo

### 病毒传播的一句话公式

- **表现**：「Jean-Claude Van Damme 在两台卡车上劈叉」= 9 个字。这支广告的病毒性在概念阶段就确定了——不是因为制作精良，而是因为「一句话就能完整描述」。广告上线后 1 周内 YouTube 播放量破 3000 万，最终超过 1 亿——几乎全部来自社交分享而非付费媒体。
- **品牌目的**：Volvo Trucks 的受众不是大众消费者（你不会买一台重型卡车），而是卡车司机/物流公司。传统广告投放无法精确触达这个人群。病毒传播的解决方案：做一支每个人都会分享的广告 → 卡车司机在社交媒体上自然会看到（因为他们的朋友分享了）。
- **可迁移**：病毒「一句话公式」的测试标准：(a) 描述句 ≤ 10 个汉字或 ≤ 8 个英文单词；(b) 描述句包含一个「意外元素」——「在两台卡车上劈叉」= 意外（卡车 + 劈叉 = 不相关的两个概念）；(c) 描述句让没看过的人产生「什么？我要看！」的反应（测试：发描述句给 3 个不知情的朋友——如果没人追问 = 不成立）。Volvo Split = 9 个字 + 意外（卡车×劈叉）+ 可传播 = 病毒。
- **适用场景**：product-demo

### 极简信息密度：全片 3 个信息点

- **表现**：重复第 1 技法的要点——这条广告只有 3 个信息点。对比典型广告的信息负载（通常 10-20 个信息点：品牌名/产品名/功能1/功能2/功能3/价格/促销/渠道/日期/免责声明），Volvo Split 证明了「极简 = 极强」。
- **品牌目的**：重型卡车的购买周期是 3-5 年——不是冲动消费。广告不需要「完成交易」——只需要「建立认知」。Volvo Split 留给观众的认知只有一条：「Volvo 卡车的转向很精确」。如果观众记住了这一个信息点（并在 3 年后需要买卡车时想起），广告就成功了。如果塞了 10 个信息点但观众一个都没记住，广告就失败了。
- **可迁移**：广告信息密度的「1-3-10」法则：(a) 消费者看完后能记住的信息 ≤ 1 条；(b) 广告中的核心信息点 ≤ 3 条；(c) 广告中的任何文字/旁白/画面元素 ≤ 10 类。超过这些数字 = 信息过载 = 消费者大脑启动「忽略模式」。
- **适用场景**：product-demo, studio-ad

## 适用提示

- **最佳匹配场景**：需要「产品演示」的工业/B2B 广告、想用「极限 stunt」替代「功能列表」的创意方向、需要病毒传播的低预算高创意项目、主题关于「精确/控制/信任」的品牌故事。
- **不适用场景**：需要详细功能说明的复杂产品、没有条件做实拍 stunt 的低预算项目、受众对 stunt 表演有审美疲劳的品类（如极限运动品牌——他们的所有广告都是 stunt，需要更多叙事层次）。
- **引用优先级**：核心引用（产品演示即表演 / 极简信息设计 / 一镜到底航拍 / 音乐反差 / 病毒一句话公式）；补充参考（安全绳擦除 VFX / 对称构图 / Golden Hour 拍摄 / 频谱分离音频）。
