## 叙事技法（→ Writer）

### 留白叙事

- **表现**：全片从未让周慕云和苏丽珍接吻、上床、甚至明确说「我爱你」。两人关系的推进完全通过省略——观众看到的是一次次擦肩、在走廊里的点头、在面摊的偶遇。核心事件（两人确认配偶出轨的对话、在酒店房间共处一夜的具体内容）被故意跳过。
- **手法**：王家卫用「省略」替代「展示」。经典好莱坞会在情感高潮处给特写+音乐推进；王家卫在高潮处切黑场或切到空镜头。观众被迫用自己的想象填补空白——这种「参与式叙事」让情感更私人化。
- **可迁移**：产品广告中，不必展示产品的所有功能。只展示产品「被渴望」的那一刻——一只手的指尖触碰产品表面、一个眼神看向产品的方向——比完整的产品演示更有吸引力。品牌叙事中，留出「观众需要自己完成」的情感缺口，比填满所有信息更有效。
- **适用场景**：studio-ad, custom (art-film), product-demo

### 重复结构作为叙事引擎

- **表现**：全片建立在「重复」之上。同一段走廊走了无数次、同一个面摊出现了 5 次、同一段 Yumeji's Theme 循环了 9 次、旗袍换了 23 套但场景只有几个。每次重复都有微小的差异——周慕云和苏丽珍每次擦肩的距离越来越近、在面摊的对话从「借报纸」到「你老公喜欢吃什么」。
- **手法**：「重复 + 微变 = 情感进展」。在几乎相同的镜头构图、音乐、场景中，只改变一个字、一个眼神、一个手势，观众就能感受到巨大的情感位移。这是一种「减法叙事」——不需要新场景、新事件，只需在重复中累积细微变化。
- **可迁移**：品牌广告可以建立「重复视觉母题」——LOGO 出现的角度/环境/光线每次略有不同，形成一个系列。社交媒体短视频可以在 3-5 个视频中重复同一构图/同一动作，每次改变一个细节，让观众在刷到时产生「发现差异」的快感。
- **适用场景**：studio-ad, logo-animation, product-demo

### 未完成情感

- **表现**：全片的核心情感是「几乎但从未」。两人几乎牵手（手指在出租车后座几乎触碰，但没有）、几乎告白（「如果多一张船票，你会不会跟我走」→ 没有问出口）、几乎在一起（在柬埔寨吴哥窟，周慕云对着树洞说出秘密→但苏丽珍不在场）。影片结束在「未完成」上。
- **手法**：王家卫用「接近→撤回」的情感节奏替代了传统的「建立→冲突→解决」弧线。每个情感场景都停在「即将发生但尚未发生」的时刻——观众始终处于「差一点」的张力中。
- **可迁移**：广告可以不做「解决」——不是每个广告都需要「用了产品→问题解决→幸福生活」。可以停在「问题即将被解决的前一刻」，让观众自己完成最后一步。悬念式 CTA（Call to Action）比确定性 CTA 更有记忆点。
- **适用场景**：studio-ad, product-demo

### 时代背景隐喻（1962 香港）

- **表现**：故事设定在 1962 年的香港——一个即将发生巨变的城市和时间点。逼仄的公寓、共用走廊、房东太太的监视、邻里之间的流言——这些都是 1960 年代香港底层市民生活的真实写照。人物被困在狭窄空间里，就像香港被困在殖民地和回归之间。
- **手法**：不对时代背景做任何直接说明。没有字幕介绍「1962 年，香港」——而是通过旗袍样式、收音机里的粤曲、墙上的月份牌、邻里串门的规矩来「显示」时代。时代是「浸入式」的而非「被告知」的。
- **可迁移**：任何需要「时代感」的项目——不必用字幕/旁白说明时间地点。用 3 个标志性道具（服装/音乐/物品）就能建立时代锚点。1962 香港 = 旗袍 + 收音机粤曲 + 墙纸花纹。
- **适用场景**：custom (period piece), studio-ad

## 镜头语言（→ DP）

### 框架构图：门框/窗框/镜框的偷窥视角

- **表现**：全片约 70% 的镜头中，人物被某种「框」包围——门框、窗框、走廊的柱子、镜子的边框、楼梯栏杆。摄影机常常不在同一空间内，而是隔着门/窗/走廊的转角拍摄。观众不是「旁观者」而是「偷窥者」——我们在偷看他们的生活。
- **技术参数**：中长焦段（50-85mm），摄影机与被摄者之间永远隔着一层物理障碍（门玻璃/窗纱/走廊转角）。浅景深让前景的「框」微微虚化，但始终可见。构图原则：人物放在框的 1/3 处，框占画面 60%+ 面积，人物面积 <15%。
- **可迁移**：产品广告可以用「框内产品」制造偷窥感——让观众感觉自己是偶然发现了这个产品，而非被推销。手机竖屏视频天然就有「框」——利用屏幕边缘做「自然画框」，产品从框的边缘探入画面。
- **适用场景**：studio-ad, product-demo, custom (art-film)

### 慢快门步印 (Step-Printing)

- **表现**：影片中大量「慢动作」不是常规升格（高帧率拍摄→正常播放），而是降格——以 6-8fps 拍摄，然后每帧重复印 3-4 次到 24fps。结果是：画面有明显拖影（motion blur 拖尾）、人物动作有断断续续的「卡顿」感、但情绪浓度极高。最经典的段落：苏丽珍穿着旗袍上楼，拖影让旗袍花纹变成流动的色带。
- **技术参数**：拍摄 6fps，快门角度 360°（或 1/6s 快门速度）× 每帧印 4 次到 24fps = 每张画面曝光 1/6 秒，拖影长度为常规 24fps (1/48s) 的 8 倍。在数字时代模拟：24fps 素材 → 光流插值降到 6fps → 每帧重复 4 次 → 加 motion blur post-effect（方向模糊）。也可以用 AI 插帧工具（FlowFrames/DAIN）逆向操作。
- **可迁移**：HyperFrames 中模拟 step-printing：在关键情绪段落将帧率降到 8fps + 加 `filter: blur(2px)` directionally + `mix-blend-mode` 叠加两帧。产品广告的情绪高潮段落（产品亮相的 3-5 秒）用 step-printing 比正常慢动作更有「记忆质感」——像是「回忆」而非「观看」。
- **适用场景**：studio-ad, logo-animation, custom (art-film)

### 浅景深人物孤立

- **表现**：在狭窄的走廊/房间中，人物被极浅的景深从环境中「切」出来。前景和背景都虚化到仅剩色块，只有人物的脸/手/旗袍的一部分是清晰的。这种手法让物理上「很近」的人（仅隔 50cm）看起来「很远」——强化了两人虽然近在咫尺却无法在一起的孤独感。
- **技术参数**：大光圈（f/1.4-f/2）+ 中长焦（50-85mm）+ 近距离拍摄（1-2m）。在 HyperFrames 中模拟：CSS `filter: blur()` 分层 + 动态景深（根据人物与摄像机的「距离」参数调整模糊半径）。
- **可迁移**：产品广告中，用浅景深把产品从杂乱环境中「提取」出来——环境虚化到只剩颜色，产品边缘锐利。这比纯色背景更有「真实场景」的说服力。
- **适用场景**：product-demo, studio-ad

### 镜面反射调度

- **表现**：大量关键情感场景通过镜子/玻璃反射来拍摄，而非直接拍摄人物。苏丽珍在酒店走廊的镜中倒影、两人在餐厅通过玻璃窗的反射对话、周慕云在浴室镜子中看自己。镜面让画面同时包含「真实人物」和「影像人物」——哪一个是「真的」？这呼应了影片关于「身份」和「扮演」的主题（两人在排练「如果是我老公/老婆，会怎么说」）。
- **技术参数**：摄影机永远不对准镜子（避免穿帮），而是与镜子成 30-45° 角，拍摄镜子中的反射内容。灯光也必须避开镜面反射路线。在实践中意味着：镜子挂在侧墙，摄影机从另一侧拍，人物在镜前。
- **可迁移**：产品广告用镜面/反射：一个镜面桌面上产品的倒影、手机屏幕反射出用户的脸、汽车后视镜中的品牌 LOGO。反射 = 「多层现实」= 增加视觉密度和解读空间。
- **适用场景**：studio-ad, product-demo

### 极端低角度

- **表现**：多次使用接近地面的低角度拍摄——摄影机几乎放在走廊地板上，仰拍人物走过。这种角度让狭窄的走廊看起来更压抑（天花板压得很低），同时让人物有「纪念碑」般的重量——即使在最私密的时刻，人物也被「框」在一种庄严的仪式感中。
- **技术参数**：摄影机高度约 30-50cm（接近脚踝高度），广角镜头（24-35mm）增加透视畸变，让走廊的线条向画面中心汇聚。人物从镜头前走过时，小腿和旗袍下摆占据前景，脸退到画面深处。
- **可迁移**：产品广告的低角度 = 让产品看起来「巨大」和「重要」。手机放在地面上仰拍 = 手机像建筑一样宏伟。LOGO 演绎可以从地面仰角推上去。
- **适用场景**：studio-ad, logo-animation

### 旗袍色彩叙事

- **核心原则**：苏丽珍在片中换了 23 套旗袍，每套的颜色和花纹对应她当下的情绪状态。
  - **蓝/绿色系旗袍**（初期，高领口）：孤独、克制、符合「好太太」的社会期待。在走廊/面摊等公共空间穿着。
  - **红色系旗袍**（中期，渐低领口）：欲望被唤醒。在酒店/约会场景穿着。花纹变成大朵花卉——从「小碎花=规矩」到「大花=释放」。
  - **金色/黄色系旗袍**（后期）：回忆、怀旧。在最后几次见面穿着。金色=「已经回不去的黄金时刻」。
  - **深色/黑色系旗袍**（结局）：哀悼。去新加坡前、去吴哥窟前穿着。花纹消失，只剩纯色。
- **可迁移**：产品广告中可以用「产品颜色随场景情绪变化」的策略——同一产品在 3 个场景中呈现 3 种配色，每种配色=一种情感状态。服装/美妆广告直接迁移旗袍色彩叙事。
- **适用场景**：studio-ad, product-demo

### 墙纸花纹的视觉密度

- **表现**：片中的墙纸花纹极其繁复——大朵花卉、几何重复、高密度纹样。这不仅是「还原 1960 年代」，更是叙事工具：(a) 花纹的密度=情感的窒息感——花纹越密，角色越被困；(b) 花纹让画面即使在「空镜头」时也充满视觉信息，不无聊；(c) 人物穿着素色/纯色旗袍站在花纹墙前——人与环境的「图案密度差」让人物更突出。
- **可迁移**：在 AI 生图（ComfyUI/Flux）的 prompt 中添加 `ornate wallpaper, high-density floral pattern, 1960s Hong Kong interior` 可复现这种质感。HyperFrames 中可用重复 SVG pattern 做背景。
- **适用场景**：studio-ad, custom (period piece)

## 特效语言（→ VFX）

### 数字步印模拟

- **表现**：原片用光学印片机做 step-printing，每帧重复印 4 次。数字时代可用 After Effects (CC Wide Time + Posterize Time) 或 DaVinci Resolve (Speed Warp + Motion Blur) 模拟。AI 工具中，可用 FlowFrames 做逆向帧插值。
- **实现思路**：
  1. 原始 24fps 素材 → 光流分析提取运动向量
  2. 将帧率降到 6fps（每 4 帧取 1 帧）
  3. 将选出的帧复制 4 次回到 24fps 时间线
  4. 对每帧加 directional motion blur（方向=运动向量方向，长度=运动向量 × 8）
  5. 输出：有明显拖影 + 断断续续卡顿 + 高情绪浓度的画面
- **可迁移**：ComfyUI 提示词：`step printing effect, 6fps motion blur, dreamy blur trail, Wong Kar-wai style, slow shutter drag`。HyperFrames：用 CSS `filter: blur(3px)` + requestAnimationFrame 控制帧率降到 8fps + canvas 叠加前一帧的残影（opacity 0.3）。
- **适用场景**：custom (art-film), studio-ad

### 雨夜霓虹反射

- **表现**：片中 4 场雨戏，霓虹灯光在湿漉漉的街道路面上形成长长的彩色反射。雨=情感的释放（两人唯一一次接近身体接触是在雨中）、霓虹=都市中人造的「温暖」、水面反射=现实的扭曲镜像。
- **实现思路**：实拍：在湿地面 + 霓虹灯管 + 柔焦镜头。数字：在 Blender/C4D 中做 wet ground shader (roughness 0.1 + metallic 0) + area light 放置在地面上方 → 反射自然出现。AI 生图：`wet asphalt street at night, neon reflection on wet ground, Hong Kong 1960s, rain, Wong Kar-wai color palette`。
- **可迁移**：任何「雨夜霓虹」场景只需 3 个元素：(1) 湿地面做镜面反射、(2) 霓虹灯管做点状光源、(3) 大光圈散景把远处灯光变成圆形光斑。HyperFrames 中用 CSS `background: linear-gradient` 做地面反射 + `box-shadow` 做霓虹 glow。
- **适用场景**：sci-fi, studio-ad, custom (art-film)

### 烟雾氛围层

- **表现**：室内场景全部有淡淡的烟雾（香烟的烟、热水的水蒸气、香炉的烟）。烟雾的 3 个功能：(a) 柔化光线——让硬光变成柔光，人物的皮肤质感更柔和；(b) 增加空气的「可视性」——让光线本身成为画面元素（丁达尔效应）；(c) 时间感——烟雾缓慢飘动 = 时间流逝的可视化。
- **实现思路**：实拍：烟机（hazer，不是 smoke machine——hazer 产生均匀的薄雾）。数字：在 AE 中用 fractal noise + 缓慢位移做体积雾。AI 生图：`volumetric haze, cigarette smoke, light rays through smoke, atmospheric particles, cinematic atmosphere`。
- **可迁移**：棚拍广告在背景加一层 hazer → 光线的「质感」立即提升 2 个档次。HyperFrames 中用 CSS animated gradient + canvas 粒子做飘浮尘埃。
- **适用场景**：studio-ad, sci-fi, custom (art-film)

## 声音设计（→ Sound Designer）

### 音乐锚点：Yumeji's Theme 的循环使用

- **表现**：Shigeru Umebayashi 的「Yumeji's Theme」是一首 3/4 拍华尔兹，以大提琴和小提琴对话为结构。全片循环使用约 9 次，每次出现在两人「相遇/擦肩/同行」的段落。音乐的功能不是「配乐」，而是「情感锚点」——当音乐响起，观众立刻知道：这是属于他们两个人的时刻。
- **声音层次**：音乐 (70%) > 环境音 (25%) > 对白 (5%)。在 Yumeji's Theme 播放时，环境音被压到背景，对白几乎完全消失（或用字幕替代）。音乐是「第三位主角」。
- **可迁移**：品牌广告为产品指定一首「锚点音乐」——每次产品亮相时都是同一段音乐/同一个音色。观众在别处听到这段音乐时，会条件反射地想起这个品牌。AI 视频的 BGM 也可以学习这种「重复使用同一主题」而非「不断换音乐」的策略。
- **适用场景**：studio-ad, logo-animation, product-demo

### 环境音优先

- **表现**：对白不是最重要的声音。雨声、脚步声（高跟鞋在木地板上的节奏）、打字机的机械声、收音机里的粤曲——这些环境音被推到前景或与对白同等音量。这让观众「感觉」到了 1962 年的香港，而非只是「听到」故事。
- **声音层次**：环境音 (40%) > 音乐 (35%) > 对白 (20%) > 音效 (5%)。脚步声和雨声被刻意放大，让狭窄空间中的「距离」变得可听——周慕云和苏丽珍在走廊里的远近，通过脚步声音量的变化来传达。
- **可迁移**：AI 视频生成中，不要只用「音乐铺满」。加入环境音层（雨声/风声/城市背景音）可以 10 倍提升「沉浸感」。产品广告中，产品的机械声音（镜头对焦的咔嗒声/引擎的轰鸣/键盘的打字声）比旁白更有说服力。
- **适用场景**：sci-fi, studio-ad, custom (art-film)

### 对白的省略和延迟

- **表现**：关键对话常常被音乐/环境音「盖过」。周慕云说「如果多一张船票，你会不会跟我走」——这句台词的前半句被 Yumeji's Theme 淹没，观众只看到嘴型。在酒店房间里，两人的对话被墙的另一侧传来的麻将声打断。对白总是在「即将完整但被打断」的状态。
- **可迁移**：广告配音不需要每句话都清晰可辨。某些关键词（品牌名、核心卖点）清晰，其余可以用音乐/音效「半掩」——让观众「努力去听」比「喂到嘴边」更有参与感。ASMR 风格的广告尤其适用。
- **适用场景**：studio-ad, product-demo

## 适用提示

- **最佳匹配场景**：需要「未完成的美感」「东方式含蓄」「怀旧质感」的项目。旗袍/服装品牌的叙事广告、雨夜/霓虹/狭窄空间的氛围片、需要「重复结构」的品牌系列视频。
- **不适用场景**：快节奏信息型广告（APPLE-DB 模式）、需要明确 CTA 的行动驱动广告、搞笑/轻松的社交短视频。
- **引用优先级**：核心引用（色彩叙事 + 框架构图 + 音乐锚点，三者同时需要时）；特定技法借用（仅需 step-printing 效果时，只看镜头语言段；仅需旗袍色彩叙事时，只看色彩/美术段）。
