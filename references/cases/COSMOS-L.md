## 叙事技法（→ Writer）

### 环境叙事：世界先说，角色后说

- **表现**：开场 2 分钟没有角色出现。观众首先看到的是一个异星草原——紫色天空、巨型风车、无边草地。然后是一棵孤独的树。然后是一根绳子挂在树上。然后是一只羊走向绳子。环境已经把故事讲了一半——「这是一个美丽但孤独的地方，有一个生命想要离开」。
- **手法**：环境不是背景——环境是第一叙事者。在角色开口/行动之前，环境已经建立了世界观、情绪基调、核心冲突的暗示。关键：环境细节必须具有叙事功能（绳子 = 自杀暗示，风车 = 这个世界的「科技」，草地 = 孤独的无限性）。
- **可迁移**：任何新世界/新场景的建立——先展示环境（10-30 秒），让环境「说话」。环境细节选择原则：每个展示的环境元素必须具有双重功能（视觉美感 + 叙事信息）。
- **适用场景**：sci-fi

### 非人类主角的情感共鸣

- **表现**：主角是一只羊（Franck）——没有人类面孔、没有语言、没有手。但观众在 2 分钟内就与他建立了情感连接。为什么？因为他的动作传达了明确的情绪：走向绳子的犹豫（自杀暗示）→ 被龙卷风卷走的恐惧 → 在洗衣店的困惑 → 对 Victor 的好奇。
- **手法**：非人类角色的情感传达依赖 3 个替代通道：
  1. **身体语言**：羊的步伐速度（慢 = 悲伤，快 = 恐惧）、耳朵位置（下垂 = 沮丧，竖起 = 好奇）
  2. **眼睛**：虽然羊的眼睛不能做人类的表情，但 Blink 频率和注视方向传达了情绪（长时间不眨眼 = 呆滞/抑郁，快速眨眼 = 困惑，注视新事物 = 好奇）
  3. **环境反应**：角色对环境的反应告诉观众角色的内心。Franck 注意到洗衣机的反应（后退半步 = 恐惧，靠近一步 = 好奇）让观众理解他的心理。
- **可迁移**：设计非人类/机器人/外星人角色时，优先开发「身体语言词典」——每个情绪对应哪个身体部位如何运动。不需要人类面孔——需要可读的身体动作。
- **适用场景**：sci-fi

### 无对白情感的渐进建立

- **表现**：全片前 5 分钟没有对白。Franck 的情绪完全通过动作+环境传达。当 Victor（法国口音的洗衣店老板）出现时，对白也只有少量（且是单向的——Victor 对 Franck 说话，Franck 只能回以羊叫）。真正的情感连接发生在他们在星空中飞行的段落——完全无对白，只有画面。
- **手法**：情感曲线：
  - 0:00-2:00：孤独（环境建立）
  - 2:00-3:00：恐惧（被龙卷风卷入）
  - 3:00-5:00：困惑（落入洗衣店）
  - 5:00-7:00：好奇/紧张（遇见 Victor）
  - 7:00-9:00：接受/信任（洗衣机飞行段落）
  - 9:00-10:08：希望（飞向新世界）
  整条曲线在无对白的情况下完成。
- **可迁移**：先设计情感曲线，再把每个情绪段落的「传达方式」列出来（身体动作/环境/音乐/色彩）。如果某段情绪只能用对白传达——说明视觉设计不够。
- **适用场景**：sci-fi, product-demo

## 镜头语言（→ DP）

### 自然光模拟的 CG 革命

- **表现**：开场异星草原的光线——不是「CG 灯光」，是真正的自然光模拟。太阳光穿过大气层的瑞利散射、草叶上的次表面散射（subsurface scattering）、云层的柔和阴影——这些效果在 Blender Cycles 渲染引擎中通过物理正确的光路追踪实现。
- **技术参数**：
  - 渲染引擎：Blender Cycles（无偏路径追踪器）
  - 天空模型：Hosek-Wilkie 天空模型（物理模拟的大气散射）
  - 主光源：单一方向光（模拟太阳）+ 环境光（模拟大气散射）
  - SSS（次表面散射）：草叶/皮肤/布料均开启 SSS，使光线在材质内部散射（这是「自然感」的关键）
  - AO（环境光遮蔽）：自动计算物体之间的柔阴影
  - 渲染采样：每像素 500-1000 采样（电影级品质）
- **可迁移**：
  - 3D 渲染中，「自然感」不是靠更多灯光——是靠物理正确的材质和光传输。开启 SSS、AO、全局光照。
  - ComfyUI 生图提示词：`natural lighting, subsurface scattering, god rays through atmosphere, soft shadows, photorealistic meadow, golden hour light, volumetric atmosphere, path traced render`。
- **适用场景**：sci-fi, product-demo

### 开篇巨物尺度对比

- **表现**：镜头从极高处俯拍异星草原——巨大的风车（约 200m 高）在广袤的草地上。然后镜头逐渐下降，降了约 30 秒才到地面层——Franck 只有几厘米大。风车与羊的尺度对比 ≈ 2000:1。
- **技术参数**：
  - 机位：从 300m（虚拟高度）开始俯拍 → 缓慢下降到 2m（地面层）
  - 运动：30 秒的缓慢下降，速度约 10m/s
  - 构图：下降过程中 Franck 始终在画面中央（或接近中央），保持视觉焦点
  - 景深：下降过程中逐渐从深景深（f/16，一切清晰）过渡到浅景深（f/2.8，聚焦 Franck）
- **可迁移**：开篇巨物→微物的尺度下降是一种强力叙事工具。用「大」（环境）对比「小」（角色），让观众在视觉上感受到角色的渺小和脆弱——然后再用故事展示他的勇气。
- **适用场景**：sci-fi

### 微距材质凝视

- **表现**：在洗衣店内部，摄影机多次极近距离凝视物体表面——Victor 的胡须毛发（每根独立渲染）、洗衣机的金属表面（划痕+指纹+反射）、Franck 的羊毛（每根纤维的 SSS 散射）。这些「无叙事功能」的凝视镜头传达了「这个世界的真实感」。
- **技术参数**：
  - 微距镜头等效：100mm 微距，f/2.8（极浅景深）
  - 材质细节：毛发系统（粒子毛发，每根独立）+ 金属材质（GGX 着色器 + 粗糙度贴图）+ 羊毛（毛发粒子 + 密度贴图）
  - 每个微距凝视持续 2-4 秒（足够长让观众「感受」材质，足够短不打断叙事）
- **可迁移**：在 CG 作品中插入 2-3 个「材质凝视」镜头——展示材质真实性的「证据」。这些镜头放在角色动作之间的呼吸间隙。产品演示同理——插入 2-3 个材质微距特写，让观众「感受」产品品质而非「被告知」。
- **适用场景**：product-demo, sci-fi

### CG 虚拟摄影机的「物理感」

- **表现**：虽然全片是 CG（没有真实摄影机），但摄影机运动遵循了真实摄影机的物理约束——有惯性、有微小的晃动、有推拉时的呼吸感。摄影机从不做「不可能的运动」（如穿过固体物体）。
- **技术参数**：
  - 运动模拟：为虚拟摄影机添加微小的 noise 动画（模拟手持）
  - 加速度约束：摄影机加速度 ≤ 真实摄影机物理上限（dolly ≤ 3m/s², 摇臂 ≤ 1m/s²）
  - FOV 选择：24mm（广角）/ 35mm（标准）/ 85mm（长焦），严格模拟真实镜头焦段
- **可迁移**：做全 CG 作品时，给虚拟摄影机添加「物理约束」——惯性、晃动、焦段限制。观众潜意识能感知到「违反物理的摄影机运动」，即使他们说不出来为什么不对。
- **适用场景**：sci-fi

### 世界观/场景设计要点

- **「暖色科幻」**：Cosmos Laundromat 打破了科幻=冷色调（蓝/灰/黑）的刻板印象。它的科幻世界以暖色（橙/棕/金/绿）为主——这是一个「有人情味」的科幻世界。
- **材质的历史感**：洗衣店的一切都有使用痕迹——磨损的木地板、生锈的洗衣机、褪色的海报。这个世界不是「新造的」——它是有历史的。历史感 = 真实感。
- **「从牢笼到自由」的色彩旅程**：草原（绿+紫，美但被困）→ 龙卷风（灰蓝，危机）→ 洗衣店（暖橙，安全中转）→ 星空（深蓝+金，自由）。色彩跟随叙事的情绪弧线移动。
- **非对称设计**：洗衣店的内部故意设计得不对称——门不正、窗户大小不一、物品摆放不整齐。对称 = 人工/计算机，不对称 = 自然/人类。

## 特效语言（→ VFX）

### 材质渲染的物理真实性

- **表现**：Cosmos Laundromat 是 Blender Foundation 为展示 Cycles 渲染引擎能力而制作的作品。每个材质都基于物理着色器——金属使用 GGX 微面元模型、玻璃使用折射 + 菲涅尔效应、布料使用次表面散射 + 纤维模拟。
- **实现思路**（技术要点）：
  - 金属：GGX 着色器 + 粗糙度贴图（划痕和指纹通过 roughness map 实现）
  - 玻璃：折射率 1.45（真实玻璃）+ 色散（chromatic dispersion，仅 2-3 个镜头）
  - 布料/羊毛：毛发粒子系统（每根纤维独立渲染）+ 次表面散射（SSS）
  - 体积光：体积散射着色器模拟光线穿过灰尘/雾气
  - 渲染采样：电影级 500-1000 samples/pixel
- **可迁移**：
  - Blender/Cycles：开启 Filmic 色彩管理（更大的动态范围）+ Denoising（降噪）
  - ComfyUI 提示词：`photorealistic fur rendering, GGX metallic shader, subsurface scattering on skin, volumetric god rays, physically based rendering, path traced, 4K quality`
  - HyperFrames：无法直接做 3D 渲染。但可导入预渲染的材质视频层 + CSS 合成。
- **适用场景**：sci-fi, product-demo

### 自然光的光传输模拟

- **表现**：异星草原的光线效果完全依赖 Cycles 的全局光照 + 物理天空模型——没有手动添加任何「特效灯光」。太阳方向、大气密度、云层厚度——调整这些参数，光线自动正确传播。
- **实现思路**：
  - 天空模型：Hosek-Wilkie（Blender 内置，模拟真实大气散射）
  - 全局光照：Cycles 路径追踪（光线在场景中弹射 12-16 次）
  - 体积光：当光线穿过体积材质（雾/尘/云）时自动计算散射和吸收
  - 无需手动补光——全局光照自动处理间接照明
- **可迁移**：
  - 任何 3D 软件（Blender/Unreal/Unity）：优先使用物理天空 + 全局光照，减少手动补光。物理正确的光照 = 更少的灯光调试时间 + 更好的结果。
  - AI 生图：在提示词中指定 `natural lighting, global illumination, path traced, physically based sky model, volumetric atmosphere, indirect lighting bounce`。
- **适用场景**：sci-fi

### 体积光效的情绪运用

- **表现**：
  - 洗衣店内部：阳光从窗户斜射入，穿过空气中的灰尘形成可见的光柱（god rays）。灰尘粒子在光柱中缓慢飘动。
  - 龙卷风：体积云+体积雾组合，光线在云层内散射。
  - 星空飞行：体积星云（volumetric nebula）——星际尘埃在远处形成彩色光雾。
- **实现思路**：Blender 的体积着色器（Volume Scatter + Volume Absorption）放在包围整个场景的体积框中。密度和颜色根据场景情绪调整——洗衣店 = 低密度暖色，龙卷风 = 高密度灰色，星云 = 极低密度彩色。
- **可迁移**：
  - ComfyUI 提示词：`volumetric god rays, dust particles in sunlight, volumetric fog, atmospheric haze, crepuscular rays through window, volumetric nebula in space`
  - 体积光的密度控制原则：情绪越「沉重」→ 密度越高。情绪越「轻盈」→ 密度越低。
- **适用场景**：sci-fi

## 声音设计（→ Sound Designer）

### 异世界环境音景

- **表现**：
  - 异星草原：风的声音是「不同的」——不是地球的风。低频的嗡鸣 + 草叶摩擦的沙沙声 + 远处风车的吱呀声。这些声音告诉观众「这不是 Kansas」。
  - 洗衣店：洗衣机运转的机械声（多台洗衣机的不同转速形成自然的节奏）、水声、蒸汽声、木地板的嘎吱声。
- **声音层次**：环境音 (50%) > 音乐 (30%) > 音效 (15%) > 对白 (5%)。环境音被推到前景——让观众首先「听到」世界，然后才看到角色。
- **可迁移**：为异世界设计声音时，从真实世界的声音开始（风/水/机械）→ 改变 pitch（±3 semitones）和混响（RT60 调大 2-3x 制造「大空间」感）→ 叠加不常见的 texture（低频嗡鸣/金属共鸣/珊瑚礁声音采样）→ 得到「熟悉但不同」的异世界声音。
- **适用场景**：sci-fi

### 音乐的情感引导

- **表现**：音乐是交响乐风格，但配器克制——不是 100 人交响乐团，而是小型合奏（弦乐四重奏 + 钢琴 + 少量管乐）。音乐的情感轨迹精确匹配 Franck 的情感曲线：
  - 孤独段：独奏大提琴，缓慢下行旋律
  - 恐惧段：低音提琴 tremolo + 不和谐弦乐
  - 飞行/自由段：全乐团齐奏，主旋律上行
- **可迁移**：AI 视频 BGM 设计——先画出情感曲线，在曲线上标注情绪（孤独/恐惧/希望），为每段情绪匹配乐器编制和旋律方向。音乐的最小单位不是「一首歌」——是「一个情绪段落」。
- **适用场景**：sci-fi

### 音效的材质叙事

- **表现**：每个材质有专属的声音——羊毛（柔软的摩擦声）、金属洗衣机（低频嗡鸣 + 高频震动）、木地板（嘎吱 + 空洞的脚步声）、草地（沙沙 + 风啸）。当 Franck 的羊蹄从草地走到木地板→走到金属洗衣机面板时，脚步声连续变化——材质变了，声音也变了。
- **声音层次**：材质交互声 (20%) > 环境音 (50%) > 音乐 (30%)。材质声音虽然占比不高，但它们是「触觉」的主要来源——在视觉之外给大脑提供「我在触摸这个东西」的幻觉。
- **可迁移**：设计「材质声音地图」——场景中的每个表面都有对应的脚步声/接触声。角色在场景中移动时，声音跟随材质变化。这是低成本提升沉浸感的最有效方式之一。
- **适用场景**：sci-fi, product-demo

## 适用提示

- **最佳匹配场景**：全 CG 科幻/奇幻项目、需要展示材质/渲染能力的作品、非人类主角的故事、需要「暖色科幻」视觉风格的创作。
- **不适用场景**：快节奏动作片、实拍为主的创作、需要大量对白的叙事。
- **引用优先级**：核心引用（自然光模拟 / 材质渲染 / 环境叙事 / 非人类主角设计 / 暖色科幻色彩方案）。
