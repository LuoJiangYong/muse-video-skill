## 使用工作流

1. 用户提出某品类创作需求 → Agent 查本清单找匹配候选
2. 确认收录候选 → 用 B站/YouTube 链接下载视频
3. `ffprobe` 检查编码（H.264 才能 video_analyze）
4. 按 `_TEMPLATE.md` 标准拆解
5. 更新 INDEX.md 全表
6. git commit + push
