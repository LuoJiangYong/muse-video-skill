# 棚拍广告 — Studio Ad

> **场景特征**：受控环境下的品牌视觉叙事。灯光、构图、道具、演员调度——一切都可控，每个像素都是设计出来的。
> **角色激活优先级**：DP（灯光+机位）> Art Director（色调+道具）> Director（审核节奏和信息传达）。

---

## 场景特征

| 维度 | 描述 |
|------|------|
| **空间** | 受控棚内 / 有限场景数（通常 1-3 个棚景） / 可完全操控灯光 |
| **时间** | 15-90s / 精确到帧的节奏控制 |
| **核心目标** | 传递品牌信息 + 创造记忆点（不是讲完整故事） |
| **信息载体** | 画面 70% + 旁白/对白 20% + 文字/LOGO 10% |

### 关键差异化参数（vs 其他场景）

| 参数 | 棚拍广告独有 | 其他场景对比 |
|------|------------|------------|
| **灯光可控度** | 100%（棚内光线完全可设计） | sci-fi 靠 CG 模拟，product-demo 常用自然光 |
| **机位选择** | 360° 自由（无空间限制） | product-demo 受限产品角度，sci-fi 靠 CG 虚拟摄影机 |
| **演员调度** | 核心要素（有真人角色） | product-demo 通常只有手/操作者，sci-fi 以 CG 角色为主 |
| **道具** | 全部可控，是设计元素 | sci-fi 全 CG，product-demo 以产品本身为唯一道具 |
| **节奏** | 精确到帧（广告标准） | sci-fi 由叙事驱动，LOGO 演绎由动态驱动 |

## 灯光方案

> 棚拍广告的灯光是最重要的技术决策。ART DIRECTOR 定色调方向，DP 在此模板上实现。

### 机位-灯光模板

```
棚拍场景通用机位布局：

        背景（可更换背景板/绿幕）
        ┌──────────────────┐
        │                  │
        │   [产品/演员]    │
        │                  │
        └──────────────────┘
         ↑               ↑
      [Rim Light]    [Rim Light]
     (后侧上 45°)    (后侧上 45°)
                   
                   
    [Key Light] ————→ 被摄物 ←———— [Fill Light]
   (侧前 30-45°)       ↑         (对侧，柔光)
                   
                   [摄影机]
              (正前方或微侧)
```

### 灯光配置模板

```json
{
  "lighting_setup": {
    "style": "减法灯光 / 三点布光 / 高调 / 低调",
    "key_light": {
      "type": "大型柔光箱 (4x6ft) 或 蝴蝶布+聚光灯",
      "position": "被摄物侧前 30-45°，略高于被摄物",
      "color_temp": "5600K（日光）或 3200K（暖调）",
      "intensity": "100%"
    },
    "fill_light": {
      "type": "反光板 或 低功率柔光箱",
      "position": "被摄物对侧，略低于 key light",
      "intensity": "25-50%（控制反差比）"
    },
    "rim_light": {
      "type": "小型聚光灯 + 蜂巢",
      "position": "被摄物后侧上 45°，左右各一个",
      "intensity": "50-75%",
      "note": "作用：分离主体与背景。深色产品必加，白色背景可省略"
    },
    "background_light": {
      "type": "散射光 / 彩色 LED 洗墙",
      "purpose": "均匀照亮背景 或 创造渐变效果",
      "note": "如果背景是品牌色，需要与品牌色精确匹配"
    },
    "practical_lights": [
      "画面内的光源（台灯/屏幕/霓虹灯管），作为光源动机"
    ]
  }
}
```

## 机位模板

### 产品广告标准机位

| 机位 | 角度 | 景别 | 功能 | 占全片比例 |
|------|------|------|------|-----------|
| **Hero Shot** | 正面微俯 | 中景-特写 | 产品最佳角度，最重要的镜头 | 25-30% |
| **45° 展示** | 45° 侧前方 | 中景 | 展示产品的厚度和侧面设计 | 15-20% |
| **细节特写** | 微距正前方 | 大特写/插入 | 材质、工艺、关键功能 | 20-25% |
| **使用场景** | 正面/微侧 | 中景-全景 | 产品在「被使用」中 | 15-20% |
| **品牌落版** | 正面居中 | — | LOGO + 产品 + CTA | 5-10% |

## 演员调度模板

```json
{
  "talent": {
    "count": "1-3 人（少即是多——棚拍广告不是群像剧）",
    "roles": [
      {
        "type": "产品使用者",
        "action": "使用/操作/展示产品",
        "focus": "手和产品的关系，脸不是重点",
        "wardrobe": "中性/简约，不抢产品注意力"
      },
      {
        "type": "品牌代言人",
        "action": "直接对视镜头，传递品牌信息",
        "focus": "面部表情 + 眼神 + 肢体语言",
        "wardrobe": "品牌色调或对比色"
      }
    ],
    "direction_notes": [
      "演员的视线方向：看产品 > 看镜头 > 看远方。看镜头的时长 ≤ 20%",
      "动作速度比正常慢 20%（广告视觉需要「被看清」）",
      "避免快速眨眼/吞咽等小动作（特写会被放大）"
    ]
  }
}
```

## 角色参数

> 以下参数为各角色在棚拍广告场景下的默认配置。Agent 在激活对应角色时，需同时参考此处的场景特定参数。

### 灯光策略

| 参数 | 值 |
|------|-----|
| 主光源策略 | 大柔光源（柔光箱/蝴蝶布），干净清爽 |
| 反差比 | 1:2-1:4 |
| 色温 | 5600K 日光平衡 |

### 情绪曲线

| 参数 | 值 |
|------|-----|
| 曲线形状 | 慢起→渐强→峰终 |
| 关键特征 | 0-40% 铺垫，40-80% 加速，最后 20% 直冲品牌 |

### 音效密度

| 参数 | 值 |
|------|-----|
| Foley 密度 | 中（产品操作关键） |
| Ambience 重要性 | 中 |
| Design SFX 用量 | 低 |
| UI 音效 | 低 |

## 常见陷阱

| 陷阱 | 避免方法 |
|------|---------|
| 灯光太多太杂（棚拍的典型病） | 先定 1 个主光源，其他只补充。参考 BR2049 的减法灯光 |
| 机位全是正面 | 45° 展示面至少占 15%，细节特写至少占 20% |
| 道具喧宾夺主 | 背景道具的饱和度必须低于产品色 |
| 演员表演过度 | 动作速度比正常慢 20%，眼神看产品的时间 > 看镜头 |
| 产品在画面中太小 | Hero Shot 中产品应占画面 40-60% |
