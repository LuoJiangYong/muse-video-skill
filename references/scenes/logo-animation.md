# LOGO 演绎 — Logo Animation

> **场景特征**：3-10 秒的纯粹视觉冲击。LOGO 是唯一的主角——材质、动态、光线、声音全部服务于「让这个标志被记住」。
> **角色激活优先级**：VFX（材质+粒子+动态）> Art Director（品牌色适配）> Sound Designer（音效节奏）。

---

## 场景特征

| 维度 | 描述 |
|------|------|
| **空间** | 无物理空间——纯抽象/图形空间。LOGO 可以悬浮在虚空中 |
| **时间** | 3-10s（极短）。3s=标准，5s=有叙事，8s=有故事 |
| **核心目标** | 让 LOGO 的出现方式本身成为品牌记忆 |
| **信息载体** | 画面 100%。无对白、无旁白、极少文字（LOGO 本身就是文字） |

### 关键差异化参数（vs 其他场景）

| 参数 | LOGO 演绎独有 | 其他场景对比 |
|------|-------------|------------|
| **唯一焦点** | LOGO 占画面 100%，没有配角 | 其他场景都有环境/角色/产品等多个焦点 |
| **时间极短** | 3-10s，每一帧都有重量 | studio-ad 30-90s，有时间展开 |
| **材质即内容** | LOGO 的材质本身就是信息（玻璃→通透，金属→坚固，液态→流动） | 其他场景材质是背景/辅助 |
| **动态节奏** | 不是叙事节奏，是视觉-听觉的同步节奏 | studio-ad 是信息节奏，sci-fi 是叙事节奏 |
| **品牌色定律** | 品牌色必须精确匹配，色差不可容忍 | 其他场景品牌色可灵活运用 |

## 材质分类

### 材质-品牌关联

| 材质 | 品牌联想 | 动态建议 | 音效建议 |
|------|---------|---------|---------|
| **玻璃** | 通透、精致、高端、科技 | 折射光扫过 + 缓慢旋转 | 高频叮咚声 + 玻璃共鸣 |
| **金属（抛光）** | 坚固、可靠、经典、权威 | 镜面反射闪光 + 正面推入 | 低频金属共鸣 + 嗡鸣 |
| **液态金属** | 流动、变革、未来、AI | 从液态汇聚成形 | 液体流动声 + 金属定型声 |
| **粒子汇聚** | 数据、网络、分布式、科技 | 无数粒子从四面汇聚成 LOGO | 粒子沙沙声 → 汇聚后的低频 |
| **霓虹灯管** | 潮流、夜生活、创意、年轻 | 逐段点亮 + 轻微闪烁 | 电流滋滋声 + 霓虹嗡鸣 |
| **光线描边** | 速度、能量、运动、竞技 | 光束沿 LOGO 线条运动 | whoosh + 高频 zing |
| **水墨** | 传统、艺术、东方、收藏 | 墨迹晕开成形 | 毛笔刷纸声 + 水滴声 |
| **火焰** | 激情、能量、重生、热度 | 火焰燃烧成形 → 稳定 | 火焰 crackle + 低频 roar |

### 材质选择决策

```
品牌核心价值是什么？
├─ 科技/创新 → 玻璃 / 液态金属 / 粒子 / 全息
├─ 传统/经典 → 金属（抛光/拉丝） / 石材 / 木材
├─ 速度/运动 → 光线描边 / 火焰 / 金属
├─ 奢华/高端 → 抛光金属 + 玻璃 / 金色粒子
├─ 年轻/潮流 → 霓虹 / 故障 / 液态
├─ 艺术/文化 → 水墨 / 手绘 / 定格
└─ 自然/环保 → 粒子（叶/水）/ 木材 / 光影
```

## 动态节奏

### 三阶段节奏模型

```
3-5s LOGO 演绎的标准节奏：

阶段 1：触发（0-30%）— 什么引起了 LOGO 的出现？
  ├─ 光扫过画面
  ├─ 粒子从边缘飘入
  ├─ 线条开始流动
  └─ 混沌 → 秩序

阶段 2：成形（30-70%）— LOGO 本体如何出现？
  ├─ 粒子汇聚成形
  ├─ 线条逐渐勾勒
  ├─ 金属凝固/液态汇聚
  └─ 从模糊到清晰

阶段 3：确立（70-100%）— LOGO 如何「落地」？
  ├─ 最后一道光扫过 → 定住
  ├─ 材质最终显示（加最后一次反射/高光）
  └─ 留 1-2s 静止展示（让观众看清）
```

### 动态参数速查

| 动态类型 | 描述 | 适用材质 | 参数范围 |
|---------|------|---------|---------|
| **汇聚** | 元素从四面八方汇聚到中心 | 粒子、液态、光线 | 汇聚速度：0.3-0.5s，先快后慢 |
| **浮现** | 从模糊/透明到清晰/不透明 | 玻璃、全息 | 浮现时间：1-2s，ease-in-out |
| **生长** | 线条从起点到终点逐渐出现 | 光线描边、霓虹 | 生长速度：0.2-0.4s 每段 |
| **旋转** | LOGO 绕轴旋转至正面 | 金属、玻璃（3D LOGO） | 旋转：90°-360°，Ease-out |
| **展开** | 从压缩/折叠状态展开 | 任何 3D 材质 | 展开时间：0.5-1s |
| **脉冲** | 短暂放大缩小（强调） | 确立后的强调 | 脉冲：±5%，1-2 次，0.2s/次 |
| **巨物缓动** | 极慢的持续运动（BR2049 风格） | 巨型 LOGO、品牌分量 | 持续 3-5s，几乎看不出在动 |

### 节奏与品牌匹配

| 品牌调性 | 节奏 | 动态类型 | 音效 |
|---------|------|---------|------|
| **快速/运动** | 快（总时长 3-4s） | 光线描边 + 脉冲 | 快节奏 whoosh + 撞击 |
| **沉稳/权威** | 慢（总时长 6-8s） | 巨物缓动浮现 | 低频嗡鸣 + 金属共鸣 |
| **优雅/高端** | 中速（总时长 5-6s） | 玻璃旋转 + 光扫过 | 高频叮咚 + 弦乐短句 |
| **科技/未来** | 中速（总时长 4-5s） | 粒子汇聚 + 全息 | 数字音效 + glitch |
| **创意/艺术** | 不规则 | 水墨/手绘 | 有机声音 + 留白 |

## 品牌色适配规则

> ART DIRECTOR 负责色调（详见 `references/roles/art-director.md`）。此处聚焦 LOGO 演绎独有的品牌色精确匹配。

### 品牌色处理规则

```
1. 如果用户提供了品牌手册 → 严格使用品牌色 hex 值，不调整
2. 如果用户只说「红色」→ 选择该色系中最适合材质的那个红（金属用深红，霓虹用品红）
3. 背景色 = 品牌色的互补色或中性色（黑/白/深灰），不能抢 LOGO
4. 如果 LOGO 本身多色 → 背景必须纯色（黑/白）
5. 如果 LOGO 本身单色 → 背景可以有渐变或暗纹（但饱和度低于 LOGO）
```

### LOGO 与背景的对比度检查

```
LOGO 颜色 vs 背景颜色 → 对比度建议：

浅色 LOGO → 深色背景（黑/深灰/深蓝黑）
深色 LOGO → 浅色背景（白/浅灰/米白）
彩色 LOGO → 黑色背景（最安全）
白色 LOGO → 深色有纹理的背景（不能纯黑否则没层次）

特例：霓虹 LOGO → 极暗背景（几乎全黑），让霓虹发光效果最大
```

## 角色参数

> 以下参数为各角色在 LOGO 演绎场景下的默认配置。Agent 在激活对应角色时，需同时参考此处的场景特定参数。

### 灯光策略

| 参数 | 值 |
|------|-----|
| 主光源策略 | 单一强背光 + 暗前景（剪影感），或全亮 |
| 反差比 | 1:8（剪影）/ 1:1（全亮） |
| 色温 | RGB 自由 |

### 情绪曲线

| 参数 | 值 |
|------|-----|
| 曲线形状 | 持续上升 |
| 关键特征 | 0-100% 单调递增（能量累积型） |

### 音效密度

| 参数 | 值 |
|------|-----|
| Foley 密度 | 无 |
| Ambience 重要性 | 低 |
| Design SFX 用量 | 高（材质音效 + whoosh） |
| UI 音效 | 低 |

### 视觉策略

| 参数 | 值 |
|------|-----|
| 空间布局 | LOGO 居中绝对构图，占画面 50-70%；背景极深黑 or 品牌纯色——无环境/无场景/无叙事；前景可选：粒子/光线从画外飞入框住 LOGO |
| 深度策略 | 2.5D——LOGO 主体 3D 材质有厚度感，粒子/光线提供微弱 Z 轴暗示；或纯 2D 平面 |
| 核心道具类型 | LOGO（唯一主角）→ 粒子/光线/材质纹理（配角） |
| 色彩策略 | 品牌色 ±10°；黑白金银作为辅助；≤2 色（极简克制） |
| 材质方向 | 材质即叙事——金属拉丝/液态金属/粒子汇聚/磨砂玻璃/全息/霓虹灯管；单场景一个材质主方向，不混合超 2 种 |
| 视觉重心优先级 | ① LOGO 中心 → ② 材质动态（汇聚/展开/发光）→ ③ 无（3-10 秒无时间看第三层） |
| 空间纯度 | 零杂讯——画面中除了 LOGO + 材质动态 + 粒子外，什么都不存在 |

## 常见陷阱

| 陷阱 | 避免方法 |
|------|---------|
| LOGO 出现后立刻消失 | 确立阶段留 1-2s 静止展示——观众需要看清 |
| 动态太快看不清 | 总时长 ≥ 3s。每个阶段的运动至少 0.5s |
| 材质与品牌不匹配 | 用材质-品牌关联表检测：你的材质在传递什么信息？ |
| 品牌色偏差 | 如果用户提供了品牌色 hex，严格匹配。背景不能偏色 |
| 音效与动态不同步 | 音效的 hit point 必须精确对齐 LOGO 的成形瞬间 |
| 过度设计——LOGO 变得无法辨认 | LOGO 的可读性是最高优先级。所有动态和材质服务于 LOGO，不掩盖 LOGO |
