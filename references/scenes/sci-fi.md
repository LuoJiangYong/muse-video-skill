# 科幻设定 — Sci-Fi

> **场景特征**：构建一个不存在的世界，并让观众相信它。核心挑战不是「做得多炫」，而是「世界的规则感」——科技、环境、社会必须自洽。
> **角色激活优先级**：Art Director（世界观构建）> VFX（科技特效语言）> DP（巨物美学+特殊灯光）> Writer（世界观驱动的叙事）。

---

## 场景特征

| 维度 | 描述 |
|------|------|
| **空间** | 虚构世界——城市/星球/太空站/虚拟空间。无物理限制，全凭想象力 |
| **时间** | 不限 / 通常 60s+（世界观需要时间建立） |
| **核心目标** | 让观众相信这个世界的存在 + 在其中推进故事 |
| **信息载体** | 环境/氛围 50% + 角色 25% + 科技/特效 15% + 对白 10% |

### 关键差异化参数（vs 其他场景）

| 参数 | sci-fi 独有 | 其他场景对比 |
|------|------------|------------|
| **世界是虚构的** | 所有视觉元素都需要「发明」并保持自洽 | 其他场景基于现实世界，只需美化 |
| **科技一致性** | 必须有科技规则，不能随意（今天全息投影明天复古打字机，除非这是设定） | 产品演示只需要展示产品科技 |
| **异化美感** | 「不像地球」是核心美学目标 | studio-ad 追求「更美的地球」，product-demo 追求「清晰可见」 |
| **粒子氛围必要性** | 粒子是营造「活的世界」的必须元素 | 棚拍广告粒子是可选的，产品演示通常不用 |
| **尺度游戏** | 巨物 vs 微物的尺度对比是 sci-fi 标志性视觉 | 其他场景尺度保持日常比例 |

## 世界观规则模板

> ART DIRECTOR 负责世界观构建（详见 `references/roles/art-director.md` 世界观构建模板）。此处聚焦 sci-fi 场景独有的科技规则和一致性检查。

### 科技水平声明（必填）

```json
{
  "tech_level": {
    "era": "近未来(20-50年) / 远未来(100-500年) / 极远未来(1000年+) / 另类历史 / 平行宇宙",
    "defining_tech": "这个世界最关键的 1-3 个技术（如：全息投影 + AI 意识上传 + 反重力）",
    "tech_visibility": "科技外显（到处是屏幕和线缆）/ 科技内隐（看起来像自然）/ 混合",
    "power_source": "所有科技的能源从哪来？",
    "transportation": "人怎么移动？",
    "communication": "人怎么沟通？",
    "medical": "人怎么治疗/延长寿命？"
  }
}
```

### 科技一致性检查清单

> 每个 sci-fi 项目必须通过此清单。Agent 在构建世界观时逐项核对。

- [ ] **能源自洽**：所有科技有统一的能源逻辑（不能说「飞行器用核能，但手机还是锂电池」）
- [ ] **信息载体一致**：全息投影无处不在？还是 AR 眼镜？还是纸质？选一个主导形式
- [ ] **没有「魔法科技」**：每个科技元素必须有世界内的解释（不需要说原理，但要说「因为 ___ 所以 ___」）
- [ ] **建筑与科技匹配**：如果世界有反重力技术 → 建筑不应该是盒子形；如果世界能源匮乏 → 建筑应该低矮/地下
- [ ] **角色与科技的关系**：角色习惯了科技（不惊叹），还是科技是稀缺的（角色敬畏）？
- [ ] **没有无意中的复古**：除非是设定（另类历史），否则不能今天全息投影、明天打字机

## 异化美感参数

### 「不像地球」的视觉策略

| 策略 | 实现方式 | 示例 |
|------|---------|------|
| **天空不是蓝色** | 改变天空颜色（橙色/紫色/绿色/永远黑夜/多个月亮） | BR2049 洛杉矶的尘橙天空 |
| **尺度失常** | 建筑/物体 — 极大或极小。人在画面中占比 < 5% | BR2049 的巨型建筑 |
| **地平线不同** | 弯曲地平线（太空站）/ 垂直城市（无地平线）/ 完全平坦 | COSMOS-L 的异星草原 |
| **重力异常** | 漂浮物 / 垂直水流 / 反常建筑形态 | — |
| **光不是从上方来的** | 底光为主 / 多方向光源 / 生物光 / 无影 | BR2049 华莱士总部的金色水面底光 |
| **永远有东西在飘** | 粒子氛围是必须的——灰烬/雪/尘埃/花粉/数据粒子 | BR2049 的全片粒子系统 |
| **植物是陌生的** | 人造发光植物 / 巨型变异 / 金属质感 / 无植物 | BR2049 的枯树 + 全息花 |

### 异化美感强度控制

```
强度 1（轻度异化）：地球的未来版。天空还是蓝的，但建筑更高，多一点霓虹。
  → 适用：近未来品牌广告、可穿戴设备

强度 2（中度异化）：明显不是地球，但参考了地球。天空是橙色的，建筑是倒金字塔。
  → 适用：赛博朋克广告、创意短片

强度 3（重度异化）：完全陌生的世界。天空是紫色的，重力只有一半，植物会发光。
  → 适用：纯科幻短片、概念片、世界构建练习
```

## 常见陷阱

| 陷阱 | 避免方法 |
|------|---------|
| 世界观不一致（今天全息明天打字机） | 用科技一致性检查清单逐项核对 |
| 过度依赖「霓虹=赛博朋克」 | 赛博朋克不只霓虹——关键是「高科技低生活」的对比 |
| 忽略粒子氛围 | sci-fi 世界需要粒子让环境「活起来」——每个场景至少 1 种粒子 |
| 科技太完美（全息投影无瑕疵） | 学习 BR2049——全息有扫描线/色彩偏移/闪烁 |
| 画面太干净（没有生活的痕迹） | sci-fi 世界需要粗糙感——锈迹/灰尘/涂鸦/磨损 |
| 角色对科技不惊讶 | 除非角色第一次见，否则科技是日常——不需要每次都展示「惊叹表情」 |
